## 分发规则

### 微信生态裂变机制

小程序剧 的分发核心在于微信生态裂变(群分享 / 朋友圈 / 公众号导流 / 视频号 联动),与抖音 / 快手的「算法漏斗」模型完全不同。各信号权重排序(2026-Q2 公开运营数据推算,*estimated*):

- **群分享转化(~30% *estimated*)** —— 用户分享到微信群后,群成员点击打开的比例。门槛约 5-10%(远低于短视频平台的转发率,但单次转化深度高)。
- **朋友圈曝光(~25% *estimated*)** —— 用户分享到朋友圈的曝光转化,门槛 ~2-5% 点击率。
- **公众号导流(~20% *estimated*)** —— 通过公众号文章嵌入小程序卡片导流,转化率 8-15%(高质量定向流量)。
- **视频号 联动(~15% *estimated*)** —— 视频号短视频挂载小程序入口,2024-Q3 后成为主要获客通道。
- **搜索流量(~5% *estimated*)** —— 微信搜一搜 关键词匹配,需做好小程序名称与描述的 SEO。
- **会话留存(~5% *estimated*)** —— 已观看用户在微信「最近使用」中的回访。

### 长剧集模式与画幅

小程序剧 与抖音 / 快手的最大差异是「长剧集模式」:

- **单季集数**:典型 10-30 集(抖音 / 快手短剧通常 3-10 集完结);长篇可达 50-80 集。
- **单集时长**:1-3 分钟(短集)或 5-10 分钟(长集);长集模式在小程序剧转化率优于短集。
- **画幅**:9:16 竖屏(主流);部分精品短剧采用 16:9 横屏 + 9:16 竖屏双版本。
- **字幕**:必须内嵌;字幕安全区与抖音 / 快手相同,但需避开微信小程序底部导航栏(高度 ~98px)。
- **封面**:小程序卡片封面 5:4(推荐 800×640),与视频号封面 9:16 共存,需准备两套素材。

### 小程序入口与承载要求

- **小程序入口**:视频号短视频挂载、公众号文章嵌入、群分享卡片、搜一搜结果、微信「发现 → 小程序」列表。
- **承载要求**:小程序需通过微信审核(类目:视频 / 影视);服务器域名需 HTTPS;内容需符合《微信小程序运营规范》。
- **加载性能**:首屏加载 ≤ 3 秒(微信官方推荐);加载超时会导致跳出率上升 30%+。
- **包大小限制**:主包 ≤ 2MB,总包(含分包)≤ 20MB;长剧集需采用「云存储 + 流式加载」架构。

### 发布频率与账号等级

- **更新频率**:小程序剧无严格的「每日发布频率」限制(不同于抖音 / 快手的刷量判定),但建议每周更新 2-3 集,保持用户黏性。
- **小程序等级**:微信对小程序有「信用分」体系(0-100 分),违规会扣分;低于 60 分会限制分发,低于 40 分会下架。
- **公众号联动**:绑定公众号的小程序可获得额外导流(图文推送挂载卡片);未绑定则仅靠自然裂变。

## 内容红线

微信小程序剧 的内容红线在 cn-content-rules.md §红线 8 类通用红线之外,有 5 项平台专属高发风险(2026-Q2 微信安全中心公告):

### 平台专属红线

| 风险项 | 等级 | 触发场景 | 关联 cn-content-rules.md |
|--------|------|----------|--------------------------|
| 诱导分享 | 🔴 | 「分享到 3 个群解锁下一集」「不分享就封号」等强制分享话术 | §虚假宣传 |
| 多级分销 | 🔴 | 拉人头返佣、层级奖励、「邀请好友赚 ¥X」机制 | §虚假宣传 |
| 涉政 比短视频平台严 | 🔴 | 任何涉及国家领导人、台海 / 游行 / 体制议题;微信审核比抖音 / 快手更严 | §政治敏感 |
| 诱导关注 / 误导跳转 | 🟡 | 假按钮(「点击领取」实为关注公众号)、伪装系统提示 | §虚假宣传 |
| 内容擦边(微信标准更严) | 🔴 | 微信对色情低俗判定比抖音 / 快手更严;短剧中的接吻镜头超过 3 秒、床戏暗示均会被拒 | §色情低俗 |

### 小程序剧专属审核要点

- **暴力血腥**(🟡):打斗、流血镜头需做剪影处理;近景刀具、枪械禁止;微信审核比抖音 / 快手更严。
- **色情低俗**(🔴):微信对衣着暴露、亲密戏判定最严;女频豪门虐恋题材的「床戏暗示」「接吻 > 3 秒」是高发风险,需用侧拍 / 剪影 / 虚化处理。
- **民族宗教**(🔴):涉及民族风俗、宗教仪式、地域歧视的内容,微信审核比抖音 / 快手更严。
- **政治敏感**(🔴):任何涉及国家领导人、台海 / 港澳 / 体制议题直接下架,且小程序账号会被封禁。
- **版权侵权**(🟡):BGM 必须使用微信曲库或自有版权;影视剧片段混剪需 ≤ 30 秒并标注来源。
- **未成年人大额消费**(🔴):诱导未成年人充值超过单笔 ¥50 或月累计 ¥200,微信会主动介入退款并处罚小程序。

### 风险等级与处置流程

- 🟢 安全:正常分发。
- 🟡 风险:机器初审限流,人工复审 12-48 小时(微信复审周期长于抖音 / 快手)。
- 🔴 违规:立即下架;累计 3 次 🔴 → 小程序封禁 7-30 天 + 信用分扣 20 分;累计 5 次 → 永久下架 + 主体账号受限。

## 付费机制

### 小程序内购(微信支付)

小程序剧 的付费通过微信支付完成,在微信生态内闭环,无需跳转外部。具体模式与阈值(*estimated*):

- **首免费集数**:通常前 5-10 集免费(长剧集模式,免费集数多于抖音 / 快手的 3-5 集),用以建立情感投入。
- **单集价格区间**:¥3-10 / 集(主流);¥1-3 / 集(新人补贴);¥10-30 / 集(精品长剧)。整体高于抖音 / 快手 ¥0.5-5。
- **整季打包**:¥39-199 / 整季(10-30 集);¥199-499 / 长季(50-80 集)。
- **会员订阅**:小程序剧会员 ¥29-99 / 月,可观看该小程序所有剧集;部分小程序采用「点数制」(充值 ¥100 得 120 点,1 集扣 1 点)。

### 付费卡点 设计(小程序剧核心)

小程序剧的 付费卡点 设计是变现关键,根据 _shared/glossary.md「付费卡点」条目,典型规则如下:

- **卡点 集数**:每 3 集至少 1 个 付费卡点;10 集剧至少 3-5 个 卡点(行业最低标准)。
- **卡点 位置**:每集 60-80% 处设置悬念收尾(cliffhanger),吸引观众付费解锁下一集。
- **卡点 强度**:卡点 必须是「剧情节转折点」(揭穿阴谋、身份反转、生死关头),禁忌平淡收尾。
- **付费转化率**:行业平均 3-8%(*estimated*),即每 100 个看到卡点的用户,3-8 个会付费解锁;精品短剧可达 10-15%。

### 季票 / 全集 解锁

- **季票**:一次性支付解锁整季(¥39-199),适合 binge watching 用户;季票折扣通常 30-50%(相比单集累计)。
- **全集解锁**:长剧集(50+ 集)提供全集解锁(¥199-499),适合深度粉丝。
- **预售**:未完结剧集可预售「剩余集数包」,提前回笼资金;但需明确退款规则(若剧集烂尾,需部分退款)。

### 分账模型(小程序剧特有)

小程序剧的分账模型与抖音 / 快手不同,主要因为微信支付是底层通道而非内容分发平台:

- **微信支付通道费**:0.6%(标准费率;部分行业 0.38%)。
- **小程序主体(版权方)收入**:扣除通道费后 100% 归小程序主体(若小程序直接运营短剧)。
- **第三方分发平台抽成**:若小程序接入「短剧分发平台」(如某些 MCN 提供的 SaaS),平台抽成 10-30%;创作者 / 版权方收入 70-90%。
- **公众号 + 小程序联动**:公众号文章导流到小程序,公众号主体与小程序主体可不同(分账由双方协商)。
- **无平台分账**:与抖音 / 快手的「平台抽成 25-30%」不同,微信生态更接近「自运营」模式,版权方收益比例更高。

### 充值与提现

- **充值方式**:微信支付(唯一);最低充值 ¥6。
- **提现周期**:T+1 到 T+7(取决于小程序主体结算设置);提现手续费 0.6%(微信支付通道费)。
- **退款规则**:已解锁集数原则上不退款;违规下架时系统自动退还;未成年人大额消费微信会主动介入退款。

## 备案要求

### 双重备案(小程序剧特有)

小程序剧 与抖音 / 快手最大的差异是「双重备案」—— 既是短剧(广电总局),又是小程序(微信平台 ICP)。任一缺失都会导致下架(2026-Q2 状态):

| 备案类型 | 触发条件 | 备案主体 | 关联 cn-content-rules.md |
|----------|----------|----------|--------------------------|
| 广电总局 网络剧 备案 | 同抖音 / 快手(集数 / 时长 / 商业分发触发) | 短剧版权方 | §备案要求 §X.1-2 |
| AI 漫剧 备案 | 全 AIGC 内容,任一集公开发布 | 短剧版权方 | §备案要求 §X.3 |
| 微信小程序 ICP 备案 | 小程序上线即触发(无内容门槛) | 小程序主体企业 | §备案要求 §X.5 |
| 增值电信业务许可证 | 小程序涉及付费内容分发(¥X 万 / 月营收 *estimated*) | 小程序主体企业 | §备案要求 §X.6 |

### 备案号 展示位置(小程序剧专属)

小程序剧 需展示双重 备案号,违反任一会下架:

- **视频内嵌**:每集片头 5 秒内或片尾字幕区,显示广电网剧备案号 文本(字号 ≥ 视频高度的 2.5%)。
- **小程序设置页**:`小程序后台 → 基本设置 → 主体信息`,显示 ICP 备案号 与增值电信业务许可证号。
- **小程序「关于」页**:用户可点击「关于本小程序」查看备案信息。
- **备案号 格式**:使用占位格式如 `<广电网剧备案号:二类-XXXX-XXXX>` 与 `<ICP备案号:京ICP备XXXXXXXX号>`(*示例*,禁止编造真实备案号 字符串,威胁模型 T-01-07 缓解)。

### AIGC 标识 要求

微信对 AIGC 标识 的要求与抖音 / 快手基本一致(网信办 2025-09-01 统一规则),但有小程序专属补充:

- **显式标识**:视频右下角持续显示「AI 生成」角标(尺寸 ≥ 视频高度的 3%)。
- **隐式标识**:视频文件需嵌入 C2PA / Content Credentials 元数据。
- **文本披露**:剧情简介需包含「本剧部分内容由 AI 辅助创作」声明;小程序「关于」页需补充 AI 内容使用说明。
- **小程序类目**:含 AIGC 内容的小程序需在类目中标注「AI 内容生成」(2025-Q4 微信新增类目)。
- **触发条件**:任一帧含 AIGC 元素即触发。

详见 cn-content-rules.md §AIGC 标识。

## 爆款公式

小程序剧 的爆款公式与抖音 / 快手有显著差异,核心在于「长剧集 + 悬念卡点 + 情感共鸣」,男频 / 女频 二元不如抖音明显。以下为 2026-Q2 观察到的高转化模板(*estimated*):

### 长剧集悬念公式(小程序剧主流)

| 题材 | 公式 | 卡点 频率 | 完播率 目标 |
|------|------|-----------|-------------|
| 重生复仇(长篇) | 被害 + 重生 + 步步为营 + 多线反派 + 终极反转 | 每 3 集 1 个 卡点 | ≥ 50%(单集) |
| 豪门恩怨(长篇) | 灰姑娘 + 霸总 + 婆媳 + 商战 + 多角恋 | 每 3 集 1 个 卡点 | ≥ 45%(单集) |
| 穿越重生(长篇) | 现代人穿越古代 + 利用现代知识 + 宫斗 + 称霸 | 每 3 集 1 个 卡点 | ≥ 48%(单集) |
| 战神归来(长篇) | 退役 + 隐藏身份 + 家人受辱 + 步步揭开 + 终极对决 | 每 3 集 1 个 卡点 | ≥ 45%(单集) |

**核心要素**:
- 前 3 秒钩子:开场即悬念(神秘电话、突发事件、身份反转)。
- 多线叙事:长剧集允许 2-3 条平行叙事线,禁忌单线平铺。
- 卡点 节奏:每 3 集 1 个强 卡点(付费转化关键);单集 60-80% 处设置小 卡点。
- 字幕:精致小字、优雅字体(细黑体 / 宋体),强调情感台词。
- BGM:情感向弦乐、钢琴,与悬念转折同步。

### 情感共鸣公式

小程序剧的另一个核心是「情感共鸣」,超越男频 / 女频 二元:

| 题材 | 公式 | 卡点 频率 | 完播率 目标 |
|------|------|-----------|-------------|
| 亲情催泪 | 父子 / 母女 / 兄妹分离 + 误会 + 重逢 | 每 5 集 1 个 强 卡点 | ≥ 40% |
| 友情背叛 | 发小 / 同窗 + 利益冲突 + 反目 + 和解 | 每 4 集 1 个 卡点 | ≥ 38% |
| 职场逆袭 | 小职员 + 受辱 + 隐瞒身份 + 全公司打脸 | 每 3 集 1 个 卡点 | ≥ 42% |
| 师徒情 | 师父 + 徒弟 + 传承 + 背叛 + 回归 | 每 5 集 1 个 卡点 | ≥ 35% |

**核心要素**:
- 前 3 秒钩子:开场即情感冲突(争吵、离别、重逢)。
- 击中点:每集至少 1 个让观众「泪目 / 心动 / 心酸」的瞬间。
- 多集铺垫:长剧集允许慢热铺陈(前 5-10 集建立人物关系),禁忌急于付费卡点。
- 字幕:简洁、情感化、留白。
- BGM:亲情向、友情向背景音乐。

### 男频 / 女频 公式(小程序剧次主流)

小程序剧也有男频 / 女频 短剧,但需调整为长剧集模式:

- **小程序剧男频**:战神归来、赘婿逆袭、都市修仙题材适用,但需扩展为 10-30 集长篇;单集爽点 密度可略低于抖音,但每 3 集必须有 1 个大爽点。
- **小程序剧女频**:豪门虐恋、闺蜜背叛、替身白月光题材适用,但需扩展为 10-30 集长篇;情感线需更细腻、多线交错。

### 完播率优化技巧(小程序剧专属)

1. **前 3 秒钩子**:与抖音 / 快手一致。
2. **长剧集结构**:采用「主线 + 副线」双线叙事,避免单线疲劳;每 5-10 集设置「小高潮」(中段反转)。
3. **付费卡点 设计**:每 3 集 1 个强 卡点,单集 60-80% 处小 卡点;卡点 必须是剧情转折点(详见 付费机制 节)。
4. **公众号联动**:每集同步推送公众号文章(剧情解析、幕后花絮、演员访谈),增强用户黏性。
5. **社群运营**:建立微信群(老铁群 / 粉丝群),新集上线时群内推送,转化率高于自然流量。
6. **预告片**:每集结尾预告下一集亮点(15-30 秒),提升付费解锁意愿。

详见 _shared/platform-comparison.md 与 cn-content-rules.md §爆款元素交叉表。

## Drift Signals

出现以下任一信号时,触发 off-cycle re-verification(不等季度窗口):

- 微信发布新的「小程序运营规范更新」或「视频号 创作者政策」。
- 备案号 格式调整(广电总局新规,与抖音 / 快手同步)。
- 付费机制 大改(例如微信支付通道费率调整、新增订阅模式)。
- 爆款公式 趋势转变(观察 5+ 顶流小程序剧集体转向新题材)。
- 网信办 发布新的 AIGC / 短剧 相关通知。
- 微信小程序审核规则收紧(尤其涉政 / 色情低俗 判定)。
- 诱导分享 / 多级分销 判定规则变更。
- 视频号 与小程序联动入口调整。
- 未成年人保护 相关退款政策变更。
