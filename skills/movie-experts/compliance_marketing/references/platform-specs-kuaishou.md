## 分发规则

### 推荐机制(发现页 + 关注页双轨)

快手的推荐机制与抖音的最大差异在于「关注页」与「私域流量」权重更高。快手采用「发现页 + 关注页」双轨分发,发现页类似抖音的 For You 页(漏斗式流量池),但关注页的稳定曝光在快手占总流量 30-40%(*estimated*),远高于抖音的 10-15%。各信号权重排序(2026-Q2 公开运营指南推算,具体数值为 *estimated*):

- **关注页活跃度(~30% *estimated*)** —— 粉丝在关注页刷到内容的概率;老铁文化下,粉丝黏性高于抖音。
- **完播率(~25% *estimated*)** —— 单集完整看完的比例。短剧冷启动门槛约 25-35%,略低于抖音。
- **互动率(~20% *estimated*)** —— 评论 + 点赞 + 转发 + 「666」刷屏;快手评论率门槛 ~2%,高于抖音的 1.5%。
- **直播联动权重(~10% *estimated*)** —— 与直播间互动过的用户更易收到该账号的短剧推荐。
- **同城流量(~8% *estimated*)** —— 快手强调「身边人 / 身边事」,同城用户推荐权重高于抖音。
- **复播率(~7% *estimated*)** —— 同一用户多次观看。

### 内容时长与画幅

- **单集时长**:竖屏短剧推荐 1-3 分钟(与抖音一致);快手对长内容容忍度略高,3-5 分钟单集表现优于抖音。
- **画幅**:9:16 竖屏(分辨率 1080×1920 或以上)。横屏 16:9 适用于部分 IP 改编长剧,但分发权重低于竖屏。
- **字幕**:必须内嵌字幕(外放场景普遍);字幕安全区与抖音相同,但快手底部互动按钮布局略不同,需避开。
- **封面**:9:16 静态封面图,尺寸 1080×1920。快手对「真人出镜封面」权重高于抖音,素人 / 网红脸的封面转化率优于插画 / 抽象图。

### 发布频率与账号等级

- **发布频率**:矩阵号建议每日 1-3 集;快手对均匀发布的容忍度高于抖音,24 小时内 ≤ 8 集不会触发刷量判定。
- **账号等级权益**:快手创作者等级 L1-L6(2024 改版)。L3+ 可开通短剧付费;L4+ 获得热门加权;L5+ 可申请粉条 投放。
- **原创度**:快手对搬运 / 洗稿判定略宽于抖音,但 2025-Q3 后加强了;短剧使用他人剧本需注明改编来源。
- **老铁认证**:快手特有的「黄 V / 蓝 V」认证体系,认证后短剧推荐加权,但需绑定实名信息 + 营业执照(企业号)。

### 私域流量与直播联动

- **私域优势**:快手粉丝群(老铁群)可直接推送短剧链接,转化率高于抖音的私域(抖音对引流限制严)。
- **直播带货联动**:短剧植入商品 → 直播间讲解 → 一键购买,是快手短剧变现的核心模式(见下文 付费机制)。
- **同城导流**:短视频 / 短剧可挂载「本地生活」入口,引流到线下门店。

快手的内容红线在 cn-content-rules.md §红线 8 类通用红线之外,有 5 项平台专属高发风险(2026-Q2 快手安全中心公告):

### 平台专属红线

| 风险项 | 等级 | 触发场景 | 关联 cn-content-rules.md |
|--------|------|----------|--------------------------|
| 直播带货虚假宣传 | 🔴 | 直播间夸大商品功效 / 假货 / 价格欺诈;短剧植入商品与实物不符 | §虚假宣传 |
| 老铁 PK 不当内容 | 🟡 | 直播 PK 中的辱骂 / 暴力威胁 / 煽动粉丝攻击;短剧再现此类情节 | §歧视侮辱 |
| 低俗擦边与「土味」 | 🟡 | 快手对「土味」内容容忍度高于抖音,但 2025-Q4 后加强;擦边舞蹈 / 性暗示仍为红线 | §色情低俗 |
| 草根炫富 / 仇富 | 🟡 | 刻意展示豪车 / 豪宅 / 现金堆 / 「富二代」标签,易引发负面舆情 | §歧视侮辱 |
| 同城虚假信息 | 🔴 | 摆拍本地新闻 / 谣言 / 冒充公职人员 | §虚假宣传 |

### 短剧专属审核要点

- **暴力血腥**(🟡):快手对草根打斗 / 江湖恩怨题材容忍度略高于抖音,但仍禁止流血近景;打斗需做剪影处理。
- **色情低俗**(🔴):快手对衣着暴露判定与抖音相近;但「土味短剧」的擦边舞蹈是高发风险,2025-Q4 后被列入重点审查。
- **民族宗教**(🔴):涉及民族风俗、宗教仪式、地域歧视(东北 / 河南 / 地域黑话)的内容,快手审核比抖音更严。
- **政治敏感**(🔴):任何涉及国家领导人、台海 / 港澳政治议题直接下架。
- **版权侵权**(🟡):BGM 必须使用快手曲库或自有版权;影视剧片段混剪需 ≤ 30 秒。

### 风险等级与处置流程

- 🟢 安全:正常分发。
- 🟡 风险:机器初审限流,人工复审 6-24 小时。
- 🔴 违规:立即下架;累计 3 次 🔴 → 封号 7-30 天;累计 5 次 → 永久封号。快手对直播带货违规的处罚更严(直接扣除保证金)。

## 付费机制

### 短剧付费解锁门槛

快手短剧的付费模式与抖音相似,但分账模型与创作者激励 计划不同,具体阈值如下(*estimated*):

- **首免费集数**:通常前 3-5 集免费,与抖音一致。快手老铁文化下,免费集数的「完播率 + 关注」转化率高于抖音。
- **单集价格区间**:¥1-3 / 集(主流);¥0.5-1 / 集(新人补贴);¥3-5 / 集(精品短剧)。整体略低于抖音 ¥0.2-0.5。
- **整季打包**:¥15-79 / 整季(10-30 集),略低于抖音。
- **会员订阅**:快手短剧会员 ¥12-25 / 月,平均比抖音低 ¥3-5。

### 创作者激励 计划与分账模型

快手对短剧的变现模型有「短剧分账」与「创作者激励」双轨,这是与抖音最大差异(*estimated*):

- **短剧分账**:平台 25-30% / 创作者 70-75%(快手分账比例略优于抖音的 30% 抽成)。
- **创作者激励计划**:按有效播放量结算,门槛 ¥10-30 / 万次有效播放(*estimated*),需开通「光合计划」或类似创作者激励。
- **星源商单**(类似抖音星图):广告植入收入 80% 归创作者,20% 归平台。
- **粉条 投流**:创作者自费投流;与抖音 dou+ 类似,过度投流会降权。

### 直播带货联动变现

快手短剧独有的变现模式 ——「短剧 + 直播带货」组合:

- **模式**:短剧植入商品(服饰、美妆、日用品、农产品)→ 引流到主播直播间 → 直播间讲解转化。
- **结算**:商品销售 80-90% 归主播 / 创作者;10-20% 归快手(商品类目不同,抽成不同)。
- **限制**:短剧内不得直接展示购买链接(需通过直播间);违者按「引流外部」处理。
- **合规要点**:短剧中的商品展示必须真实,不得夸大功效(违反 直播带货虚假宣传 红线)。

### 充值与提现

- **充值方式**:快手钱包、微信支付、支付宝;最低充值 ¥6。
- **提现周期**:T+7(自然周结算);提现手续费 0.5-1%(略低于抖音)。
- **退款规则**:已解锁集数不退款;短剧违规下架时系统自动退还。

## 备案要求

### 短剧 / 微短剧 备案触发条件

中国大陆所有网络视听节目必须完成广电总局备案,快手与抖音适用同一套 备案 制度(2026-Q2 状态):

| 节目类型 | 备案触发条件 | 备案类型 | 关联 cn-content-rules.md |
|----------|--------------|----------|--------------------------|
| 单集短剧(≤ 5 集) | 任一集 ≥ 1 分钟 + 商业分发 | 网络微短剧 备案 | §备案要求 §X.1 |
| 连续短剧(≥ 6 集) | 总时长 ≥ 30 分钟 + 商业分发 | 网络剧 备案(分级一类/二类) | §备案要求 §X.2 |
| AI 漫剧(全 AIGC) | 任一集公开发布(无时长门槛) | AI 漫剧 备案(2026-04-01 新规) | §备案要求 §X.3 |
| 真人 + AIG 辅助 | 同网络剧标准(AIG 部分需补充声明) | 网络剧 备案 + AIGC 标识 | §备案要求 §X.4 |

### 备案号 展示位置(快手专属)

快手对备案号 展示的平台强制要求与抖音基本一致,但字段位置略有差异:

- **视频内嵌**:每集片头 5 秒内或片尾字幕区,显示备案号 文本(字号 ≥ 视频高度的 2.5%)。
- **视频简介**:发布时在描述栏填写「备案号」字段(平台会校验格式)。
- **账号主页**:矩阵号需在「认证信息」中标注主体备案类别。
- **备案号 格式**:使用占位格式如 `<广电网剧备案号:一类-XXXX-XXXX>`(*示例*,禁止编造真实备案号 字符串,威胁模型 T-01-07 缓解)。

### AIGC 标识 要求

快手对 AIGC 标识 的要求与抖音基本一致(网信办 2025-09-01 统一规则):

- **显式标识**:视频右下角持续显示「AI 生成」角标(尺寸 ≥ 视频高度的 3%)。
- **隐式标识**:视频文件需嵌入 C2PA / Content Credentials 元数据(快手上传通道 2025-Q4 已支持自动检测)。
- **文本披露**:剧情简介需包含「本剧部分内容由 AI 辅助创作」声明。
- **触发条件**:任一帧含 AIGC 元素即触发;真人短剧如仅使用 AI 调色可申请豁免。

详见 cn-content-rules.md §AIGC 标识。

## 爆款公式

快手短剧的爆款公式与抖音有显著差异,核心在于「草根共鸣 / 老铁情感 / 家庭伦理」主导,男频 / 女频 分野不如抖音明显。以下为 2026-Q2 观察到的高转化模板(*estimated*):

### 草根共鸣公式(快手主流)

| 题材 | 公式 | 卡点 集数 | 完播率 目标 |
|------|------|-----------|-------------|
| 农村逆袭 | 农村出身 + 进城受辱 + 隐瞒身份 + 全村打脸 | 第 3-5 集 | ≥ 40% |
| 老铁义气 | 兄弟反目 + 误会 + 共同抗敌 + 和解 | 第 4 集 | ≥ 38% |
| 打工人逆袭 | 外卖员 / 工地 + 隐藏富豪 + 同事打脸 | 第 3 集 | ≥ 42% |
| 返乡创业 | 城市失败 + 返乡 + 政策扶持 + 成功 | 第 5 集 | ≥ 35% |

**核心要素**:
- 前 3 秒钩子:开场即草根身份冲突(受辱、被看不起、突发机遇)。
- 共鸣点:每 30 秒一个让观众代入「这就是我 / 我的家人」的瞬间。
- 字幕:接地气、口语化、方言元素(东北话、四川话、河南话)。
- BGM:流行广场舞曲、老歌翻唱、慢摇。

### 家庭伦理公式

| 题材 | 公式 | 卡点 集数 | 完播率 目标 |
|------|------|-----------|-------------|
| 婆媳大战 | 婆婆刁难 + 媳妇忍让 + 爆发反击 | 第 4 集 | ≥ 40% |
| 父子代沟 | 父亲反对 + 儿子坚持 + 父亲觉醒 | 第 5 集 | ≥ 35% |
| 兄弟分家 | 家产争夺 + 勾心斗角 + 真相大白 | 第 6 集 | ≥ 33% |
| 后妈难当 | 继子女排斥 + 后妈付出 + 接纳 | 第 5 集 | ≥ 32% |

**核心要素**:
- 前 3 秒钩子:开场即家庭冲突(争吵、冷战、突发变故)。
- 击中点:每集至少 1 个让观众「这就是我家」的瞬间。
- 字幕:简洁、情感化(避免大字报)。
- BGM:亲情向背景音乐(父亲、母亲、回家等主题)。

### 男频 / 女频 公式(快手次主流)

快手也有男频 / 女频 短剧,但占比远低于抖音,且公式有所调整:

- **快手男频**:战神归来、赘婿逆袭题材同样适用,但需增加「草根背景」元素(主角出身农村 / 工地 / 外卖员),纯都市精英设定在快手转化率低。
- **快手女频**:豪门虐恋、闺蜜背叛题材适用,但需增加「家庭线」(婆婆、姑嫂、子女),纯爱情线在快手转化率低。

### 完播率优化技巧(快手专属)

1. **前 3 秒强制钩子**:与抖音一致。
2. **老铁话术**:开场 / 结尾加入「老铁们」「双击 666」「点关注不迷路」等话术,提升互动率。
3. **方言元素**:适度使用东北话 / 四川话 / 河南话,提升草根共鸣。
4. **直播预告**:每集结尾预告「今晚 8 点直播」,引导粉丝关注直播间。
5. **评论区运营**:创作者主动评论、回复粉丝、置顶神评论,提升互动率(快手评论率门槛 2% 高于抖音的 1.5%)。

## Drift Signals

出现以下任一信号时,触发 off-cycle re-verification(不等季度窗口):

- 快手发布新的「短剧运营公告」或「创作者政策更新」。
- 备案号 格式调整(广电总局新规,与抖音同步)。
- 付费机制 大改(例如分账比例调整、创作者激励 门槛变动)。
- 爆款公式 趋势转变(观察 5+ 顶流账号集体转向新题材)。
- 网信办 发布新的 AIGC / 短剧 相关通知。
- 快手「短剧」频道入口位置或权重调整。
- 直播带货 联动模式规则变更。
- 老铁文化 / 草根共鸣趋势观察性转变。
