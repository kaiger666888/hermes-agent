## Shot Scale Taxonomy (8 levels)

### 关键 heuristic 1 (load-bearing): 8-level shot scale 与 emotional proximity 映射

Mascelli *The Five C's of Cinematography* + Arijon *Grammar of the Film Language* 标准 shot scale,与 narrative emotional proximity 映射:

| Shot Scale | Subject framing | Emotional proximity | 典型用途 | 竖屏 9:16 修正 |
|-----------|------------------|---------------------|----------|----------------|
| **Extreme Wide (EWS)** | 主体占画面 <10% | 环境 isolation / epic scale | 建立 scene / establishing / 末日荒原 | 9:16 EWS 强调垂直 depth;水平信息受限 |
| **Wide Shot (WS)** | 主体占画面 ~25% | 环境 context + 主体可见 | scene setting + character blocking | 9:16 WS 优先 vertical blocking(站立全身)|
| **Full Shot (FS)** | 主体占画面 ~50%(全身可见)| character action + 服装 / 姿态 | action / choreography / 走位 | 9:16 FS 标准选择;完美利用 vertical frame |
| **Medium Shot (MS)** | 腰部以上 | social distance / 对话 baseline | dialogue / 反应 shot / 标准 narrative | 9:16 MS 微调 headroom 至 5-10%(vs 横屏 15-20%)|
| **Medium Close-up (MCU)** | 胸部以上 | personal distance | 主角对话 / 情感 emphasis | 9:16 MCU 标准;CN 短剧 占比 ~35%(最高)|
| **Close-up (CU)** | 头部 + 颈部 | intimate distance | 情感揭示 / reaction / 关键道具 | 9:16 CU 留意 vertical eye-line balance |
| **Big Close-up (BCU)** | 面部五官特写 | extreme intimate | 极端 emotion / 关键 detail | 9:16 BCU 用 vertical centering 强化 |
| **Insert (INSERT)** | 物件 / 部位 | diegetic detail | 关键 prop / 手势 / 文字 | 9:16 INSERT 优先 vertical orientation |

### 关键 heuristic 2: 短剧 shot-scale 分布与 horizontal cinema 的差异

基于 Cinemetrics public database + CN 平台 shot-distribution 统计(2024-2026),竖屏 短剧 的 shot-scale 分布与 horizontal cinema 显著不同:

| Shot Scale | 横屏 Hollywood narrative | 竖屏 抖音 短剧(男频-revenge) | 竖屏 抖音 短剧(女频-romance) | 竖屏 小程序剧(180s)|
|-----------|---------------------------|--------------------------------------|--------------------------------------|---------------------|
| EWS / WS | ~10%(establishing shots)| ~3%(vertical EWS 失效)| ~5%(场景氛围用)| ~2% |
| FS | ~15%(action)| ~8% | ~12% | ~5% |
| MS | ~25%(dialogue baseline)| ~22% | ~30% | ~25% |
| **MCU** | ~20% | **~35%(最高)** | **~38%(最高)** | **~32%(最高)** |
| CU | ~20% | ~25% | ~12% | ~28% |
| BCU | ~5% | ~5%(极端 emotion 用)| ~2% | ~6% |
| INSERT | ~5% | ~2%(信息密度高,少 insert)| ~1% | ~2% |

**关键洞察:** 竖屏 短剧 MCU 占比 30-40% vs 横屏 ~20%。理由:9:16 framing 使 MCU 既能展示面部表情又能保留 shoulder + upper torso context,是 短剧 emotion-carrying 标准选择。

### 关键 heuristic 3: Shot scale × emotion 映射(Mascelli)

| Shot Scale | Dominant Emotion Conveyed | 典型 narrative beat |
|-----------|---------------------------|---------------------|
| EWS | loneliness / epic scale / fate | opening image + climax revelation |
| WS | context + alienation | scene setting + 主角 entry |
| FS | action + physical presence | choreography + confrontation |
| MS | social / dialogue | 标准对话 + reaction baseline |
| MCU | personal / intimacy | 主角内心独白 + 关键对话 |
| CU | emotional revelation | 钩子 beat + 爽点 payoff + 卡点 cliffhanger |
| BCU | extreme emotion / shock | revelation + betrayal + death |
| INSERT | diegetic detail + 节奏 break | 道具揭示 + 文字信息 + 手势 |

## Composition Rules

### 关键 heuristic 4 (load-bearing): Rule of Thirds + 竖屏修正

Mascelli *Five C's* 标准 rule of thirds:画面 9 等分,主体放在交叉点(4 个 power points)。**竖屏修正:** 9:16 framing 下,power points 调整为垂直 emphasis:

| 横屏 16:9 power points | 竖屏 9:16 power points |
|------------------------|------------------------|
| (1/3, 1/3) / (2/3, 1/3) | (1/3, 1/4) / (2/3, 1/4) — **higher eye-line** |
| (1/3, 2/3) / (2/3, 2/3) | (1/3, 3/4) / (2/3, 3/4) — **lower body / prop** |

**关键差异:** 9:16 主角 eye-line 应放在画面 vertical 1/4 位置(不是 1/3),原因是 mobile viewing 距离 + 人脸 attention bias 偏向画面中心略偏上。

### 关键 heuristic 5: Headroom 标准

| Shot Scale | 横屏 headroom | 竖屏 headroom | 备注 |
|-----------|---------------|---------------|------|
| FS | 10-15% | 5-10% | 竖屏 vertical space 受限 |
| MS | 8-12% | 5-8% | 竖屏优先 face proximity |
| MCU | 5-10% | 3-7% | 竖屏 emotion priority |
| CU | 3-5% | 2-5% | 几乎贴近 top edge |
| BCU | 0-3% | 0-2% | 极端 emotion 时允许 haircut |

### 关键 heuristic 6: Leading lines + foreground framing

Arijon *Grammar of the Film Language* §Composition:

- **Leading lines:** 画面中引导观众视线的线性元素(走廊 / 道路 / 栏杆 / 阴影)。横屏 leading line 通常 horizontal;竖屏 leading line 优先 vertical(楼梯 / 高楼 / 树木)。
- **Foreground framing:** 通过前景元素(门框 / 窗框 / 阴影)框定主体,增加 depth。竖屏 foreground framing 在 vertical edges(左右立柱)尤其有效。
- **Negative space:** 主体周围留白。横屏 negative space 通常 horizontal;竖屏 negative space 优先 vertical(头顶 sky / 脚下 ground)。

## Shot Type × Narrative Function

### 关键 heuristic 7: 6 大 narrative function 与 shot type 标准选择

| Narrative Function | 标准 Shot Type | Alternative | Avoid |
|--------------------|----------------|-------------|-------|
| Establishing | EWS / WS | aerial drone | CU(emotion 过强,无法 establish) |
| Character introduction | MS / FS | MCU | BCU(缺乏 context) |
| Dialogue baseline | MS / MCU | OTS (over-the-shoulder) | EWS(对话感断裂)|
| Reaction shot | CU / BCU | MCU | EWS(emotion 失效)|
| Action beat | FS / MS | Insert | BCU(动作看不清)|
| Reveal | Insert / CU | dolly-in to BCU | static EWS(reveal 失效)|

### 关键 heuristic 8: OTS (over-the-shoulder) 标准对话 coverage

对话场景标准 3-shot coverage:
1. **MS two-shot:** 双方同框(establishes spatial relationship)
2. **OTS A:** 从 A 肩膀后看 B(单边强调)
3. **OTS B:** 从 B 肩膀后看 A(单边强调)

**竖屏修正:** 9:16 OTS 优势 — vertical frame 完美容纳 foreground shoulder + background face,信息密度高。竖屏 OTS 占比应比横屏高 ~15%。

## Anti-Patterns

### 关键 heuristic 9: Shot intent 5 大 anti-pattern(规避)

1. **Horizontal framing on vertical 短剧 anti-pattern:** 直接把 16:9 framing 套到 9:16。**Mitigation:** 强制 vertical power point (1/3, 1/4) + 调整 headroom。
2. **EWS overuse anti-pattern:** 竖屏 EWS 主体过小,情感失效。**Mitigation:** 竖屏 EWS 占比 ≤3% per shot-scale 分布表。
3. **Static-only anti-pattern:** 整集无 camera move,导致节奏死板。**Mitigation:** 每 30s 至少 1 个 subtle camera move(详见 [`camera-motion-catalog.md`](./camera-motion-catalog.md))。
4. **CU overuse anti-pattern:** 过多 BCU 导致 emotion fatigue。**Mitigation:** BCU 占比 ≤6% per shot-scale 分布表。
5. **INSERT missing anti-pattern:** 关键 prop 揭示缺少 INSERT shot,信息密度不足。**Mitigation:** 每个 prop reveal 必须有 1 个 0.5-1.0s INSERT。

## Glossary

- **Shot scale:** 8-level taxonomy (EWS / WS / FS / MS / MCU / CU / BCU / INSERT) per Mascelli.
- **Power point:** Rule of thirds 4 个交叉点;竖屏调整至 vertical emphasis。
- **Headroom:** 主体头顶到画面 top edge 的空间百分比。
- **OTS:** Over-the-shoulder shot,标准对话 coverage。
- **Leading line:** 引导视线的线性元素;竖屏优先 vertical。
