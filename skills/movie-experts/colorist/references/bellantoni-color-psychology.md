## The 8-Color Vocabulary

Bellantoni 的体系不沿用 HSV 色环,而是按"电影中观察到的情绪负载"将可见光谱切成 8 个主色。每个主色至少有 3 个 canonical cinematic deployment 与一位开创导演。

### 主色与 hue 范围(对应 SKILL.md CxSxZ 中的 C 维度)

| # | 主色 | Hue 范围(HSL,approx) | 主要情绪负载 |
|---|------|----------------------|-------------|
| 1 | Purple(紫)| 270-290°(深紫到品紫)| 灵性 / 神秘 / 王权(positive)或 腐败权威 / 道德灰区(negative)|
| 2 | Red(红)| 0-15° / 345-360° | 激情 / 爱 或 暴力 / 危险 |
| 3 | Yellow(黄)| 45-65° | 温暖 / 安全 / 希望 或 衰败 / 病态 / 妄想 |
| 4 | Blue(蓝)| 200-240° | 忧郁 / 失去 或 冷静 / 理性 或 神性 / 超越 |
| 5 | Green(绿)| 90-150° | 自然 / 生命 / 重生 或 嫉妒 / 中毒 / 不安(uncanny)|
| 6 | Orange(橙)| 15-45° | 温暖亲密 / 黄昏怀旧 或 危险预热 / 警示升级 |
| 7 | White(白)| —(亮度维度)| 纯洁 / 神圣 或 空白 / 死亡 / 临床 |
| 8 | Black(黑)| —(亮度维度)| 死亡 / 终结 或 神秘 / 未知 或 高雅 / 权威 |

### 主色与 canonical cinematic deployment 6 个 director-film triplets

每个主色至少有 3 个 canonical deployment 与一位开创导演。Bellantoni 在 n ≈ 100 部电影语料中蒸馏出下列代表性 triplet;每个 triplet 包含 director + film + 该色在该 film 中的情绪负载含义。

#### Purple 三重奏

- **Triplet 1:Kurosawa《乱》(1985)— 王权紫。** Kurosawa 用深紫色描绘战国霸主一文字秀虎的王权崩溃;紫色从最初的王权象征,在剧情发展中逐渐褪去,反映权力的虚妄。
- **Triplet 2:Villeneuve《Blade Runner 2049》(2017)— 腐败权威紫。** Wallace 公司总部的紫色光柱代表资本主义的腐败权威;Purple 是反派 Niander Wallace 的标志性环境色,代表超越人类道德的科技权力。
- **Triplet 3:Argento《Suspiria》(1977)— 神秘紫。** Argento 用饱和的品紫色描绘舞蹈学院,传递神秘与超自然威胁;Purple 是 giallo 流派的标志性色,代表观众无法理性解释的诡异。

#### Red 三重奏

- **Triplet 1:Almodóvar《关于我母亲的一切》(1999)— 激情红。** Almodóvar 用饱和的猩红色描绘 Manuela 的服装与巴塞罗那街道,Red = 失去与重生的激情循环。
- **Triplet 2:Coen《Blood Simple》(1984)— 危险红。** Marty 酒吧的血红色霓虹灯 + 红色 blood stains 承担 Marty 的"未被抑制暴力"视觉化身;Red 不是装饰,而是反派的延伸。
- **Triplet 3:Spielberg《辛德勒名单》(1993)— 象征红。** 全片黑白,唯一彩色是穿红衣的小女孩;Red = 无辜的生命 + 必然的死亡,是电影史上最强的"color as character"案例之一。

#### Yellow 三重奏

- **Triplet 1:Herzog《Stroszek》(1977)— 衰败黄。** Herzog 用昏黄的灯光描绘柏林边缘的移民公寓,Yellow = 病态、衰败、社会边缘的无力感。
- **Triplet 2:《爆裂鼓手》(2014)— 警示黄。** Fletcher 教练在排练室中,黄色荧光灯的过度曝光传递高压训练的危险预热。
- **Triplet 3:Jeunet《天使爱美丽》(2001)— 温暖黄。** Amélie 的巴黎公寓被 hyper-saturated 黄绿色笼罩,Yellow = 温暖、怀旧、童话感的安全。

#### Blue 三重奏

- **Triplet 1:Kieslowski《蓝》(1993)— 失去蓝。** 主角 Julie 的悲伤通过游泳池的蓝色水光、糖块的蓝色包装反复出现;Blue 不是被动反映,而是 grief 的 active agent。
- **Triplet 2:Cameron《阿凡达》(2009)— 神性蓝。** Pandora 星球的纳威人肌肤为蓝色,Blue = 神性、自然、超越人类的存在。
- **Triplet 3:Mann《盗火线》(1995)— 冷静蓝。** 洛杉矶夜景的城市灯光偏冷蓝,Blue = 专业、冷静、犯罪世界的距离感。

#### Green 三重奏

- **Triplet 1:Lynne Ramsay《You Were Never Really Here》(2017)— 不安绿。** 主角 Joe 的酒店环境使用病态的荧光绿,Green = 创伤后的精神不安、道德边界的模糊。
- **Triplet 2:《黑客帝国》(1999)— 数字绿。** Matrix 内部的代码瀑布为绿色,Green = 数字化、非真实、需要被破解的虚拟。
- **Triplet 3:Wright《赎罪》(2007)— 嫉妒绿。** Cecilia 的绿色长裙 = Briony 嫉妒的物质化象征,Green = 阶级与情感嫉妒的具身。

#### Orange 三重奏

- **Triplet 1:Makhmalbaf《午后时光》(1996)— 怀旧橙。** 老妇人在午后阳光中的黄色暖色调,Orange = 怀旧、岁月沉淀的温暖。
- **Triplet 2:《疯狂的麦克斯:狂暴之路》(2015)— 危险橙。** 沙漠的橘色色调 = 末日废土的危险预热,Orange = 资源匮乏世界中的暴力预告。
- **Triplet 3:Tarkovsky《飞向太空》(1972)— 温暖怀旧橙。** Solaris 站内 Earth-memory 段落的暖色调,Orange = 对地球家乡的温暖记忆。

#### White 三重奏

- **Triplet 1:Bergman《第七封印》(1957)— 死亡白。** Death 角色的苍白脸庞 + 白雪背景,White = 死亡的不可逃避。
- **Triplet 2:Kubrick《2001 太空漫游》(1968)— 临床白。** spacecraft 内部的过度白色,White = 临床、无菌、去人格化的科技理性。
- **Triplet 3:《大都会》(1927)— 神圣白。** 上层城市的白色花园 = 神圣、纯净、阶级优越的物质化。

#### Black 三重奏

- **Triplet 1:Kubrick《奇爱博士》(1964)— 权威黑。** Ripper 将军的黑色军装 = 权威、克制、潜在的暴力。
- **Triplet 2:《蝙蝠侠》系列 — 神秘黑。** 哥谭市的黑色雨夜 = 神秘、未知的犯罪世界。
- **Triplet 3:《七武士》(1954)— 死亡黑。** 强盗的黑色剪影 = 死亡的不可阻挡。

### 8-Color Vocabulary 使用规则

- **同一场景最多使用 1 个主色 + 1 个辅助色(2 个)**;3 个及以上主色同台 = "noisy",观众无法形成情绪锚定。
- **主色必须在场景的 ≥ 60% 画面占比中出现**(服装 / 道具 / 灯光 / 背景之一);否则该色是装饰而非主色。
- **主色切换(从一个主色跳到另一个)是 value-shift 信号**,不是装饰变化;切换必须有 ≥ 2.0s transition_duration,不允许 hard cut(详见 §Color Sequencing Rule)。
- colorist 在选 CxSxZ combination 时,先从 8-Color Vocabulary 表找到主色,再根据场景具体情绪标签从 canonical deployment 中选择参考。

## Prevalence Data

Bellantoni 在 n ≈ 100 部电影的语料中,人工标注每个场景的主导色与情绪标签,统计 prevalence(主导色在特定情绪类别场景中出现的占比)。下列百分比为本研究 ref 蒸馏后的代表性数据;完整 cross-tab 见 Bellantoni 原书 chapter 2-9。

### Prevalence 表(主导色 × 情绪类别)

| 情绪类别 | n(场景数,approx)| Purple prevalence | Red prevalence | Blue prevalence | Yellow prevalence | Orange prevalence | 备注 |
|----------|-----------------|-------------------|----------------|-----------------|-------------------|-------------------|------|
| Villain / 反派登场 / 道德灰区 | 47 | **~60%** | 18% | 12% | 5% | 5% | Purple 是反派主导色,Bellantoni 命名本 ref 标题即来自此观察 |
| Passion / 激情 / 情欲 | 38 | 8% | **~78%** | 5% | 6% | 3% | Red 是激情主导色 |
| Loss / 失去 / 哀悼 | 42 | 14% | 4% | **~67%** | 10% | 5% | Blue 是失去主导色 |
| Hope / 希望 / 新生 | 35 | 5% | 8% | 4% | **~58%** | 25% | Yellow 是希望主导色,Orange 次之(黄昏怀旧)|
| Tension / 紧张 / 危险 | 51 | 22% | **~55%** | 12% | 8% | 3% | Red + Purple 共占 77%,Bellantoni 称之为 "danger palette" |
| Intimacy / 温暖亲密 | 29 | 0% | 14% | 0% | 18% | **~62%** | Orange 是亲密主导色(温暖怀旧)|

### 关键 prevalence 数据点

**关键 heuristic 1(load-bearing):Purple 在 ~60% 反派场景出现。** 这不是装饰巧合,而是 Bellantoni 在 100 部电影语料中观察到的稳定 pattern —— Purple 在主流电影中承载"道德灰区 / 腐败权威 / 神秘反派"的情绪负载。Bellantoni 据此命名全书。colorist 在反派登场场景,应优先考虑 Purple 作为 C(主色)的 baseline 选择。

**关键 heuristic 2:Red 在 ~78% 激情场景出现,在 ~55% 紧张/危险场景出现。** Red 是唯一同时在两类情绪(passion + danger)中 prevalence > 50% 的色。Bellantoni 称之为 "Red dual function"。colorist 在激情或危险场景,Red 是 default 选择;但在 CN 内地部署时需 cross-check Red Duality(见 [`cn-audience-color.md`](./cn-audience-color.md) §Red Duality in CN,Red 在 CN 默认偏 positive 含义)。

**关键 heuristic 3:Blue 在 ~67% 失去场景出现。** 这是欧美电影中"blue = melancholy"的稳定 pattern。注意:此 prevalence 是北美/欧洲电影语料,CN 部署需 cross-check [`color-cross-cultural.md`](./color-cross-cultural.md) §Madden Country × Emotion Table(Madden 数据显示日本 51% Blue = sadness,与 Bellantoni 数据一致;但韩国仅 42%,荷兰仅 24%)。

**关键 heuristic 4:Yellow 是希望主导色(58%)**,与"Yellow = 衰败"(Triplet 1 Herzog《Stroszek》)形成 contrast。Yellow 在 Bellantoni 语料中有双面性 —— 在希望场景 = 温暖安全(58%),在衰败场景 = 病态(15%)。colorist 选 Yellow 时必须明确场景情绪是 hope 还是 decay,因两含义的 saturation 与 brightness 完全不同(hope = high saturation + high brightness;decay = low saturation + medium brightness)。

### Prevalence Data 使用规则

- **prevalence 是 prior,不是 deterministic rule** —— colorist 在无场景特殊信号时,应优先选择 prevalence ≥ 50% 的色。
- **当场景情绪标签与某色的 prevalence < 10% 时**(如 Yellow 用于 Villain = 5%),需标明 "operator convention override",并解释为何违反 prevalence。
- **prevalence 不能脱离场景上下文单独使用** —— 必须配合 cross-cultural caveat(见 [`color-cross-cultural.md`](./color-cross-cultural.md) §Madden Country × Emotion Table,Bellantoni 语料以北美/欧洲电影为主,CN 部署需 cross-check [`cn-audience-color.md`](./cn-audience-color.md) §Red Duality in CN)。
- prevalence 数据是 Bellantoni 在北美电影语料的观察,不应作为 universal truth 引用;在 SKILL.md body 引用时使用 "Bellantoni 2005 北美电影语料"限定词。

### Prevalence Data 的常见误用

**误用 1:把 prevalence 当 universal 规则。** Bellantoni prevalence 是北美/欧洲 100 部电影(1960-2003)的语料观察,不是 universal 真理。在 CN 短剧 部署时,Red 在 CN 默认含义是喜庆/吉利(prevalence ~ 55%),与 Bellantoni 北美语料(passion + danger 共占 ~ 130%)显著不同。colorist 在跨市场部署时,必须切换 prevalence 源:Bellantoni(北美/欧洲)vs [`cn-audience-color.md`](./cn-audience-color.md)(CN)vs [`color-cross-cultural.md`](./color-cross-cultural.md)(学术跨文化)。

**误用 2:只看最高 prevalence 色,忽略 secondary 色。** 一个情绪类别中,主导色 prevalence > 50% 时,secondary 色(第二高 prevalence)仍是有效信号。例如 Hope 类别中,Yellow 58% 主导但 Orange 25% 也是有效选择(黄昏怀旧)。colorist 在选 CxSxZ combination 时,应同时考虑 top-2 色,根据场景具体子情绪选 final。

**误用 3:跨情绪类别直接迁移 prevalence。** Red 在 passion 78% 不等于 Red 在 loss 78%;若 colorist 因"Red 高 prevalence"在所有场景用 Red,会得到无差别调色。每个情绪类别有独立 prevalence 表,colorist 必须按场景情绪标签查表,不跨类别迁移。

**误用 4:用 prevalence 单独决策,忽略 "color as character" doctrine。** Bellantoni 的核心不是"选高 prevalence 色",而是"颜色必须 driver 情绪节拍"。即使某色 prevalence = 80%,若 colorist 不能回答 "this color is the [character] of this scene because [emotion beat]",该选择仍不通过。

### Prevalence Data 与 短剧 部署的 cross-validation

Bellantoni 语料(1960-2003 北美电影)与 CN 短剧(2024-2026)有两个时间与文化 gap:

- **时间 gap**:1960-2003 vs 2024-2026,电影色彩审美 60 年演变(尤其 digital color grading 在 2000+ 普及后,色彩饱和度与精度大幅提升)
- **文化 gap**:北美/欧洲 vs CN,主导色-情绪联想不同(见 [`cn-audience-color.md`](./cn-audience-color.md) §Red Duality in CN)

colorist 在 CN 短剧 部署时,应优先使用 [`cn-audience-color.md`](./cn-audience-color.md) 数据;Bellantoni 数据作为 fallback(当 CN 数据未覆盖该情绪类别时)。Bellantoni 的 doctrine(8-color vocabulary / color as character / color sequencing)是 stable,但具体 prevalence 数值需 cross-check。

## Color as Character

Bellantoni 的核心 doctrine:**颜色不是装饰,而是角色**。一个场景的主导色必须 driver 一个情绪节拍(emotional beat),否则就是 "pretty picture without emotional stakes"。这是 colorist `## Role & Philosophy` 第一条 "Color is the first emotion the audience feels" 的理论 grounding。

### Color as Character doctrine 的 3 个 canonical 案例证明

#### 案例 1:Coen Bros. 《Blood Simple》(1984)—— Red 承担反派的视觉化身

红色不是"显得危险",而是承担反派 Marty 的"未被抑制的暴力"的视觉化身。每帧包含红色的密度,与 Marty 的情绪压抑程度反相关 —— 红色越多,Marty 越接近暴力爆发。

Coen 的具体手法:
- Marty 酒吧的红色霓虹灯在 Marty 压抑愤怒时(早期)只占画面 ~ 20% 面积
- 当 Marty 决定雇凶杀 Ray 时(中段),红色扩展到画面 ~ 50%
- Marty 持枪追杀的 climax 段,红色浸透整个画面 ~ 80%
- 颜色密度的渐变 = Marty 心理压力的渐变 externalization

colorist 在 AI 短剧/微电影 中应用此 doctrine:在反派复仇场景中,Red 占画面比例应随主角复仇动机强度递增;具体数值可通过 drawer 的 prompt 指令(`red_color_ratio` field)控制。

#### 案例 2:Kieslowski 《蓝》(1993)—— Blue 是 grief 的 active agent

蓝色不是"显得哀悼",而是主角 Julie 的 grief 的 active agent。Julie 试图与丈夫死后留下的所有事物切断联系,但蓝色不断 re-emerge,迫她 confront grief。蓝色不是被动反映情绪,而是主动推动剧情(agency)。

蓝色 re-emerge 的 3 个具体场景:
- 蓝色水晶灯罩(丈夫遗留物)—— Julie 试图丢弃但被找回
- 蓝色游泳池水光 —— Julie 在水中哭泣时,泳池的蓝色包围她
- 蓝色糖块包装 —— Julie 在咖啡店撕开糖块时,蓝色包装在特写中占画面 ~ 90%

Kieslowski 的 doctrine 是:颜色不是装饰品,而是 narrative driver;colorist 在 long-form 叙事中(尤其 小程序剧 长集),应将主色作为 recurring motif,而非每场景独立选色。

#### 案例 3:Almodóvar 《关于我母亲的一切》(1999)—— Red + Yellow 共同构成主角 emotional ground

红色 = 失去(儿子 Esteban 车祸死)的痛苦,黄色 = 重生(回到巴塞罗那开始新生活)的希望。两色交替出现的频率 = Manuela 心理愈合的进度。

Almodóvar 的 color sequencing:
- 影片前 1/3(马德里):Red 主导,Manuela 丧子之痛
- 中段(巴塞罗那寻父):Red + Yellow 混合,Manuela 在新关系中发现希望
- 影片后 1/3:Yellow 主导,Manuela 接受 Rosa 之子(艾滋病毒携带者)的诞生作为新生命的象征

colorist 在多集 短剧/小程序剧 中应用:每集主色切换的频率应与主角 emotional arc 同步;若剧本明确标注 value-shift markers,colorist 应在每个 marker 处 trigger 主导色切换。

### Color as Character 使用规则

- 在 SKILL.md CxSxZ combination 选择前,先回答:**"this color is the [character] of this scene because [emotion beat]"**。
- 若 colorist 不能用一句话回答上述句子,该色选择不通过 `color_intent_match ≥ 0.85` 阈值。
- Bellantoni doctrine 是 [`SKILL.md`](../SKILL.md) §Role & Philosophy 第一条 "Color is the first emotion the audience feels" 的理论 grounding。
- colorist 在 `## Workflow` step 1 "Emotion-Color Mapping" 中必须先验证 "color as character" doctrine,再选具体 CxSxZ 数值。
- 在 judge / ablation eval 中(Phase 6),"color as character"是 evaluation criteria 之一 —— 若 colorist 输出的 color_intent.json 中某 shot 缺少 "color as character" rationale,扣分。

### Color as Character 在 短剧 部署的实操示例

**示例:男频 revenge 短剧 爽点 climax 场景。** 主角在仓库角落反派,揭露隐藏身份,反派 realization 自己被 outmaneuvered。colorist 的"color as character"rationale 应该是:

> "Color is the [dominance / power-swing] of this scene because [protagonist rises from victim to dominator]. The Red + Orange warm radiance from protagonist's face + dark background = 'lit from within' power visualization (per [`cn-audience-color.md`](./cn-audience-color.md) §'Lit from Within' Trend). Red is the active agent pushing antagonist into shadow; not decorative."

这个 rationale 满足 3 个要求:(1)主色 = 角色;(2)情绪节拍明确;(3)参考了具体 doctrine 与 CN 部署源。

colorist 的输出 JSON 应包含该 rationale 字段:
```json
{
  "shot_id": "climax_001",
  "cxsz": {"C": "5-15°", "S": 0.78, "Z": 0.72},
  "color_as_character": "dominance / power-swing — protagonist rises from victim to dominator",
  "emotion_beat": "爽点 climax (per hook_retention 爽点 taxonomy)",
  "doctrine_ref": "bellantoni-color-psychology.md §Color as Character 案例 1 (Coen Blood Simple red = violence externalization)",
  "cn_adaptation": "Red 在 CN 默认 positive 含义 (per cn-audience-color.md §Red Duality);saturation within Douyin ceiling 0.75"
}
```

这个 JSON 是 Phase 6 ablation eval 的 judge 检查依据 —— 若 colorist 输出缺少 `color_as_character` 或 `doctrine_ref` 字段,扣分。

### Color as Character doctrine 的边界

Bellantoni doctrine 不是 universal 适用:
- **不适用 case 1:中性背景场景。** 如空镜头建立场景(establishing shot)、过场镜头(b-roll)。这些场景的主导色不必 driver 情绪节拍,可以是装饰性的 neutral palette。
- **不适用 case 2:多场景快速切换的 montages。** 如 Eisenstein metric montage 段(见 [`../editor/references/montage-theory.md`](../editor/references/montage-theory.md) §Metric Montage Frame Numbers),每 shot 0.04-0.12s,无法独立承载 color as character;整体 montage 段是 color sequencing 的一个节拍。
- **不适用 case 3:real-world documentary 风格。** 真实生活记录(尤其 草根 快手 短剧)的主导色受现场光线约束,不应强行 creative 调色;color as character doctrine 应 relax。

colorist 在判断是否应用 doctrine 时,需先确认场景类型;narrative 驱动场景必须应用 doctrine,non-narrative 或 documentary 场景可豁免。

## Color Sequencing Rule

Bellantoni 的 color sequencing rule:**当一部电影在中途切换主导色,该切换必须 signal 一个 value-shift**(角色价值观的转变),不是装饰风格变化。

### Color Sequencing 与 McKee Value-Shift 的联动

Color Sequencing 与 [`../screenplay/references/mckee-scene-design.md`](../screenplay/references/mckee-scene-design.md) §Value-Shift Rule(Phase 3 screenplay ref)是同一原理的不同侧面 —— McKee 说"每个 beat 必须有 value-shift",Bellantoni 说"value-shift 应该通过主导色切换来 externalize"。

具体联动:
- screenplay.emotion_curve 中每个 value-shift marker(如 "hope → despair"或"ignorance → revelation")对应 colorist 的一次主导色切换
- screenplay 在 value-shift marker 中输出 `recommended_color_shift` field(如 "Yellow → Blue"),colorist 验证 + 执行
- 若 screenplay 未提供 recommended_color_shift,colorist 应从 Bellantoni 8-color vocabulary 表 + Prevalence Data 中推理

### Color Sequencing 的 3 个案例

color sequencing rule 在长 form 叙事中尤其重要 —— 电影 90-180min、小程序剧 10-80 集长 form,主导色切换的频率与价值转变的节奏是观众感知"故事在前进"的关键 visual signal。

#### 案例 1:《Shutter Island》(2010)—— Blue(调查冷静)→ Green(uncanny,真相揭露)

前半段主导色 = Blue(调查冷静理性),后半段主导色 = Green(uncanny,真相揭露)。切换 = 主角从"调查者"到"病人"的 value-shift。Blue → Green 切换发生在主角 Teddy 第一次质疑自己记忆的瞬间,Green 的 uncanny 含义(见 §Green Triplet 1)承担了"现实崩塌"的视觉化。

#### 案例 2:《Her》(2013)—— Red + Pink(爱情沉浸)→ Yellow(关系新阶段)→ White(接受现实)

主导色 = Red + Pink(爱情沉浸)→ Yellow(主角 Theodora 与 Samantha 关系进入新阶段)→ White(接受现实)。每次切换 = 主角情感状态的 phase transition。Red + Pink 是"沉浸"的极致;Yellow 是"现实化"(从虚拟 AI 关系到意识到 Samantha 同时服务 8000 用户);White 是"接受与放下"。

#### 案例 3:《Pleasantville》(1998)—— 黑白 → 彩色 = 觉醒 / 自我解放

黑白 → 彩色 = 觉醒 / 自我解放的 value-shift,色彩本身是剧情的 metaphor。这是 color sequencing 的极端案例 —— 主导色从"无色"(黑白)渐变到"多彩",每个角色的色彩化反映他们的觉醒程度。

Pleasantville 的渐进色彩化揭示了 Bellantoni doctrine 的另一面:color sequencing 不必是"全片瞬间切换",可以是"per-character 渐进"。colorist 在多角色 短剧(尤其 女频 宫斗宅斗)中,可考虑 per-character 主导色:每角色有自己的"color signature",随角色成长而切换。

### 案例 cross-validation 与 短剧 应用

上述 3 个电影案例的共性:value-shift 是 colorist 触发主导色切换的充要条件。colorist 在 AI 短剧 部署时:

- 输入:`screenplay.emotion_curve` 的 value-shift markers
- 输出:`color_sequencing_plan` 字段,记录所有切换点 + 切换理由(对应哪个 value-shift)+ transition_duration(≥ 2.0s)
- 验证:整集切换频率 ≤ 短剧 2 次 / 小程序剧 3 次;若超出,需剧本端减少 value-shift 或合并

### Sequencing 规则的执行

- colorist 必须消费 screenplay.emotion_curve 的 value-shift markers(详见 [`../screenplay/references/mckee-scene-design.md`](../screenplay/references/mckee-scene-design.md) §Value-Shift Rule),在每个 value-shift 标记处 trigger 主导色切换。
- 切换必须有 ≥ 2.0s transition_duration(见 [`SKILL.md`](../SKILL.md) §Color Transitions),不允许 hard cut between 主导色。
- 切换频率上限:**每集 短剧 ≤ 2 次主导色切换**(否则 viewer 困惑);**小程序剧长集 ≤ 3 次**。
- 若整集 短剧 没有任何主导色切换,需检查剧本是否有 value-shift;若无 value-shift,可能剧本节奏过平。
- colorist 在 `## Workflow` step 6 "Transition Processing" 中执行此规则,输出 `color_sequencing_plan` 字段记录所有切换点。

## Drift Signals

本 ref 触发刷新的条件:

- **新理论颠覆 8-color vocabulary**:若有 2025+ 顶刊 color-psychology 论文提出与 Bellantoni 冲突的电影色彩分类(如新语料证伪 "purple = 60% villain prevalence"),需修订。
- **AI 生成电影语料加入**:若 AI 生成电影(Sora / Veo / Kling 出品)出现新的主导色-情绪映射 pattern(如 AI 电影普遍使用某种非 Bellantoni 色彩),需观察 + 补充。

任何 drift signal 触发,需在本 ref 顶部 `**Last-verified:**` 字段更新日期,并在修订章节标注 "revised YYYY-MM due to [drift signal]"。
