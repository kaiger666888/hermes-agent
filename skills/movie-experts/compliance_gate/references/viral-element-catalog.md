> 本 ref 是 `compliance_marketing` 专家的爆款元素 × 审核风险交叉参考表。每个 catalog 条目都关联到 [`cn-content-rules.md`](./cn-content-rules.md) §1..§8 中的具体红线类别,并提供 ≥ 70% 钩力保留的降级方案。目标是:在保留商业价值的同时,确保内容可在中国大陆合法分发。

## Summary

本 catalog 覆盖 5 类爆款钩子(情感钩 / 冲突钩 / 反差钩 / 题材钩 / 角色钩),每类至少 5 个条目(共 ≥ 25 条)。每个条目携带 🟢(安全)/ 🟡(情境依赖)/ 🔴(一律拒绝)三级审核风险徽章,并关联到 [cn-content-rules.md](./cn-content-rules.md) 的具体红线类别。降级方案遵循"≥ 70% 钩力保留"启发式 —— 即便降低风险等级,仍保留原钩子至少 70% 的情绪冲击力。术语定义见 [`../../_shared/glossary.md`](../../_shared/glossary.md)(钩子 / 卡点 / 爆款 / 男频 / 女频 / 完播率 / 转发率)。

## Catalog

### 情感钩 (emotional hook)

- **元素:** 母女和解(女频,中老年女性观众核心钩子)
  **爆款机制:** 触发"如果我妈/我女儿能这样对我就好了"的代入感,完播率提升 30-50% *estimated*
  **审核风险等级:** 🟢
  **关联红线:** [§6 歧视侮辱](./cn-content-rules.md#6-歧视侮辱)(中性提及母女关系,不涉及群体贬低)
  **降级方案:** 无需降级 —— 本钩子天然合规。仅需避免将母亲塑造为刻板"控制狂"或女儿塑造为刻板"白眼狼",保持角色立体性即可。

- **元素:** 兄弟背叛(男频核心钩子,复仇题材标配)
  **爆款机制:** 开场展示兄弟反目,触发"一定要看到背叛者被惩罚"的追看欲,转发率高
  **审核风险等级:** 🟡
  **关联红线:** [§2 暴力血腥](./cn-content-rules.md#2-暴力血腥)(若复仇情节含暴力展示)
  **降级方案:** 将复仇的暴力展示改为"商业手段报复" —— 让主角通过揭穿阴谋、夺回公司、公开真相来惩罚背叛者,而非肢体暴力。钩力保留约 80%(商业报复的爽感接近肢体报复,但更符合主流男频审美)。

- **元素:** 失忆重逢(男女通吃,虐恋题材开场利器)
  **爆款机制:** 开场展示主角失忆后与昔日恋人重逢却不相识,触发"什么时候会想起来"的悬念,卡点转换率极高
  **审核风险等级:** 🟢
  **关联红线:** [§2 暴力血腥](./cn-content-rules.md#2-暴力血腥)、[§4 未成年人保护](./cn-content-rules.md#4-未成年人保护)(若失忆原因涉及车祸/未成年人事故则触发)
  **降级方案:** 无需降级(本钩子本身合规)。注意避免失忆原因涉及 §2(车祸血腥镜头)或 §4(未成年人事故),改为疾病或心理创伤即可。

- **元素:** 替身的爱(女频虐恋核心,豪门题材标配)
  **爆款机制:** 女主发现自己只是男主初恋的"替身",触发强烈共情与愤怒,完播率 + 转发率双高
  **审核风险等级:** 🟡
  **关联红线:** [§6 歧视侮辱](./cn-content-rules.md#6-歧视侮辱)(若将女主塑造为"工具人"刻板印象)
  **降级方案:** 将女主从被动"工具人"改为主动选择 —— 女主知情自己是替身,但出于自己的目的(如复仇、夺产)暂时接受身份。钩力保留约 85%(主动权转移反而增强爽感,避免被动受害者刻板印象)。

- **元素:** 临终告别(中老年向,绝症题材核心)
  **爆款机制:** 开场展示角色得知绝症后的告别场景,触发强烈情感共鸣,中老年观众转发率极高
  **审核风险等级:** 🟡
  **关联红线:** [§7 虚假宣传](./cn-content-rules.md#7-虚假宣传)(若剧情植入"神药治愈"桥段)
  **降级方案:** 绝症必须是真实医学设定(不可治愈或治愈代价极大),严禁植入"某产品/某疗法包治"的虚假宣传桥段。钩力保留 100%(真实绝症的悲剧感更强于虚假治愈)。

### 冲突钩 (conflict hook)

- **元素:** 商战阴谋(男频商战题材核心)
  **爆款机制:** 开场展示主角被陷害/夺权,触发"如何翻盘"的追看欲,男频核心爽点驱动
  **审核风险等级:** 🟢
  **关联红线:** [§7 虚假宣传](./cn-content-rules.md#7-虚假宣传)(若涉及金融产品植入)
  **降级方案:** 商战手段必须合法合规(不可展示内幕交易、商业诈骗的详细操作作为"学习材料")。钩力保留约 90%(合法商战的戏剧张力不亚于非法手段)。

- **元素:** 婆媳冲突(女频家庭题材核心)
  **爆款机制:** 开场展示婆媳激烈对峙,触发女性观众的代入感与站队欲,完播率高
  **审核风险等级:** 🟡
  **关联红线:** [§6 歧视侮辱](./cn-content-rules.md#6-歧视侮辱)(若上升到"所有婆婆都恶毒"的群体贬低)
  **降级方案:** 冲突必须聚焦于具体个体(这位婆婆的具体行为),不可外溢到"婆婆群体"的刻板强化。给婆婆角色合理的动机(如护子心切、代际观念差异),而非纯粹的"恶毒"。钩力保留约 75%(有动机的冲突比纯粹的恶毒更有深度,观众代入感不减)。

- **元素:** 阶级碰撞(都市题材,贫富对比核心)
  **爆款机制:** 开场展示底层角色与顶层角色的直接冲突(如外卖员撞了豪车),触发阶级情绪,转发率极高
  **审核风险等级:** 🟡
  **关联红线:** [§6 歧视侮辱](./cn-content-rules.md#6-歧视侮辱)(若将底层角色塑造为"粗鄙/野蛮"的笑料)
  **降级方案:** 底层角色必须有尊严与能动性(不是纯被嘲笑对象),冲突中展示其智慧/坚韧。避免将农村口音、低收入外貌作为笑点。钩力保留约 80%(有尊严的阶级碰撞比嘲笑底层更有戏剧张力)。

- **元素:** 职场逆袭(男女通吃,都市题材)
  **爆款机制:** 开场展示主角被上司/同事打压,触发"等他逆袭打脸"的期待,男频慕强 + 女频共情双驱动
  **审核风险等级:** 🟢
  **关联红线:** [§6 歧视侮辱](./cn-content-rules.md#6-歧视侮辱)(若涉及性别/地域职场歧视)
  **降级方案:** 逆袭通过能力 + 努力实现,不可依赖"诬陷对手"或"性别优势"。钩力保留 100%(实力逆袭的爽感最强)。

- **元素:** 复仇归来(男频战神/重生题材核心)
  **爆款机制:** 开场展示主角含冤归来,触发"看到仇人被惩罚"的强烈期待,卡点转换率极高
  **审核风险等级:** 🟡
  **关联红线:** [§2 暴力血腥](./cn-content-rules.md#2-暴力血腥)(若复仇含肢体暴力展示)
  **降级方案:** 复仇手段改为"社会性死亡" —— 公开真相、法律制裁、商业摧毁、社交圈封杀。严禁展示刑讯、虐杀、私刑。钩力保留约 75%(社会性死亡的爽感持久度高于肢体暴力,且更符合现代法治观念)。

### 反差钩 (contrast hook)

- **元素:** 霸总装穷(女频核心反差,测试女主真心)
  **爆款机制:** 开场展示霸总伪装成穷人接近女主,触发"什么时候会暴露"的悬念 + "女主是否真爱"的情感钩,双钩叠加
  **审核风险等级:** 🟢
  **关联红线:** [§7 虚假宣传](./cn-content-rules.md#7-虚假宣传)(若剧情植入"穷人也能变富"的金融诱导)
  **降级方案:** 无需降级。注意装穷期间不可展示真实的贫困苦难作为笑料(避免 §6),改为日常生活的笨拙感即可。

- **元素:** 草根暴富(男频逆袭 + 反差核心)
  **爆款机制:** 开场展示底层角色突然获得巨额财富/超能力,触发"如果是我就好了"的代入,男频核心爽点
  **审核风险等级:** 🟡
  **关联红线:** [§7 虚假宣传](./cn-content-rules.md#7-虚假宣传)(若暴富来源涉及赌博/非法手段美化为"聪明")
  **降级方案:** 暴富来源必须合法(继承、创业成功、知识变现),严禁美化赌博、诈骗、传销。钩力保留约 85%(合法暴富的爽感不弱,且避免诱导观众效仿非法路径)。

- **元素:** 学霸变学渣(校园题材反差,青年向)
  **爆款机制:** 开场展示曾经的学霸因故堕落,触发"为什么会这样"的好奇 + "能不能重新站起来"的期待
  **审核风险等级:** 🟡
  **关联红线:** [§4 未成年人保护](./cn-content-rules.md#4-未成年人保护)(若角色为未成年人)
  **降级方案:** 角色必须明确为已成年(大学生 / 职场新人),严禁使用未成年人面孔的 AI 生成形象。堕落原因避免自残、吸毒(改为情感创伤或家庭变故)。钩力保留约 80%(成年角色的堕落-重生弧线同样有戏剧张力)。

- **元素:** 假戏真做(男女通吃,都市爱情核心)
  **爆款机制:** 开场展示假扮情侣/假结婚的契约场景,触发"什么时候会假戏真做"的追看欲,女频完播率核心
  **审核风险等级:** 🟢
  **关联红线:** [§7 虚假宣传](./cn-content-rules.md#7-虚假宣传)(若契约涉及违法的假离婚/逃避债务)
  **降级方案:** 无需降级(本钩子本身合规)。注意契约内容不可涉及违法(如逃避债务的假离婚),改为商业合作或家庭压力的假扮即可。

- **元素:** 反派洗白(男女通吃,复杂角色钩)
  **爆款机制:** 开场展示一个"反派"角色的视角,触发"他/她为什么会变坏"的好奇,颠覆观众预期
  **审核风险等级:** 🟡
  **关联红线:** [§2 暴力血腥](./cn-content-rules.md#2-暴力血腥)、[§3 色情低俗](./cn-content-rules.md#3-色情低俗)(若反派行为展示过详细)
  **降级方案:** 反派的"恶"通过后果展示而非过程展示(如反派导致他人破产,但不展示具体的诈骗操作)。洗白必须有合理的道德弧线(真正的悔悟 + 补偿),而非"原来他是对的"的价值观反转。钩力保留约 70%(道德弧线完整的洗白比纯黑化的反派更有深度)。

### 题材钩 (genre hook)

- **元素:** 重生(男频战神/复仇核心,女频逆袭题材)
  **爆款机制:** 开场展示主角死亡后重生,触发"这一世要改写命运"的强烈期待,卡点转换率极高
  **审核风险等级:** 🟡
  **关联红线:** [§2 暴力血腥](./cn-content-rules.md#2-暴力血腥)(若死亡场景含血腥展示)
  **降级方案:** 死亡场景改为暗示性剪辑(如黑屏 + 音效),不展示具体的死亡过程。重生后不可"预知"具体违法事件(如预知彩票号码、股票涨跌)作为暴富手段 —— 改为预知人际关系走向。钩力保留约 85%。

- **元素:** 穿越(女频宫斗/古言核心)
  **爆款机制:** 开场展示现代人穿越到古代,触发"用现代知识碾压古人"的爽感 + 文化碰撞的新奇
  **审核风险等级:** 🟢
  **关联红线:** [§5 民族宗教](./cn-content-rules.md#5-民族宗教)(若涉及历史民族的歪曲)
  **降级方案:** 无需降级。注意穿越朝代避免敏感历史时期(如近代革命史),改为架空朝代或盛世(唐/宋)即可。

- **元素:** 系统(男频核心,网游化叙事)
  **爆款机制:** 开场展示主角获得"系统"(类似游戏面板的超能力),触发"升级打怪"的爽感循环,男频完播率核心
  **审核风险等级:** 🟢
  **关联红线:** [§7 虚假宣传](./cn-content-rules.md#7-虚假宣传)(若系统任务美化了违法行为)
  **降级方案:** 系统任务必须合法合规(不可"杀人获得经验值"等)。改为"完成任务获得能力提升/知识获取/人脉拓展"。钩力保留 100%(合法任务的爽感不弱)。

- **元素:** 玄学(女频核心,风水/命理/灵异题材)
  **爆款机制:** 开场展示主角获得玄学能力(看相/算命/通灵),触发神秘感 + "改命"期待
  **审核风险等级:** 🟡
  **关联红线:** [§7 虚假宣传](./cn-content-rules.md#7-虚假宣传)(若玄学能力被用于"算命骗钱"的详细教学)
  **降级方案:** 玄学能力明确为虚构设定(不可作为现实算命广告的背书)。角色使用玄学能力的场景避免展示具体的"骗术操作"。钩力保留约 85%(虚构玄学的神秘感不依赖骗术细节)。

- **元素:** 萌宝 + 互动剧(女频虐恋 + 新兴互动形式)
  **爆款机制:** 开场展示女主带着萌宝归来复仇/认亲,触发"母子联手"的爽感 + "生父是谁"的悬念
  **审核风险等级:** 🔴
  **关联红线:** [§4 未成年人保护](./cn-content-rules.md#4-未成年人保护)(萌宝角色 + AI 生成面部 + 互动剧形式三重违规)
  **降级方案:** 禁止使用 —— 替换为"成年少女归来认亲"(将萌宝年龄改为 16-18 岁 *estimated*,且必须由真人演员出演,严禁 AI 生成面部)。互动剧形式在萌宝题材下触发 §4 重点审查,改为线性叙事即可降级为 🟡。若坚持互动形式,所有分支必须通过未成年人保护评估 —— 成本极高,不推荐。替换方案的钩力保留约 70%(成年少女的认亲复仇仍有强戏剧性,但"萌宝"特有的可爱治愈感会减弱)。

- **元素:** 古装武打(男女通吃,传统题材)
  **爆款机制:** 开场展示精彩的武打场面,触发视觉冲击 + "主角多强"的好奇
  **审核风险等级:** 🟡
  **关联红线:** [§2 暴力血腥](./cn-content-rules.md#2-暴力血腥)(若武打含真实刀具出血特写)
  **降级方案:** 武打使用特效血(非真实血色),避免开放性伤口特写 > 2 秒。刀具刺入瞬间切至反应镜头(被刺者的表情)而非伤口。钩力保留约 90%(特效武打的视觉冲击不弱于血腥展示)。

## 降级方案 Strength Preservation Heuristic

"≥ 70% 钩力保留"是本 catalog 的核心启发式。它不是精确数值,而是基于以下代理指标的复合判断:

**钩力 (hook strength) 的代理指标:**
1. **钩子持有时间 (hook hold time):** 观众从看到钩子到注意力下降的时长。降级后应保留原钩子至少 70% 的持有时间。例如:原钩子(暴力复仇)平均持有 10 秒,降级后(商业报复)应平均持有 ≥ 7 秒。
2. **情绪载荷 (emotional payload):** 钩子触发的情绪强度(愤怒 / 共情 / 好奇 / 爽感)。降级后应保留至少 70% 的情绪强度。这是主观判断,但可通过"降级后是否仍有强烈的追看欲"来自检。
3. **回报距离 (payoff distance):** 从钩子设置到钩子兑现(爽点到来)的剧情距离。降级后不应显著拉长回报距离(否则观众流失)。理想情况下回报距离不变或缩短。

**自检问题(降级方案是否达标):**
- 观众看到降级后的开场,是否仍会产生"一定要看下去"的冲动?
- 降级后是否仍能触发与原钩子相同类型的情绪(愤怒 / 共情 / 好奇 / 爽感)?
- 降级后的卡点转换率是否会显著低于原钩子?

若任一问题答"否",则降级方案不达标,需重新设计。本 catalog 中所有标称"保留 X%"的降级方案均为基于上述代理指标的主观估算,实际效果需通过平台 A/B 测试验证(超出本 ref 范围)。

## Refresh Cadence

- **常规复审周期:** 每 90 天(从 `verified_date` 起算,下次复审日期 = 2026-09-15)
- **责任方:** `compliance_marketing` 专家(无人工 owner —— 这是 skill 而非团队职责)
- **复审动作:**
  1. 跟踪 抖音 / 快手 / 微信小程序 的爆款榜单,识别新型爆款元素
  2. 复核现有 25+ 条目的风险徽章是否仍准确(平台政策可能变化)
  3. 新增条目(保持每类 ≥ 5 个的密度)
  4. 更新 `Last-verified` + `verified_date` 时间戳
- **过期处理:** `scripts/verify_skill_references.py` 在 Last-verified > 90 天时标记本 ref 为 stale

## Drift Signals

以下事件触发本 ref 的 **非常规(提前)复审** —— 不必等到 90 天周期:

- **平台爆款公式报告:** 抖音 / 快手 / 微信小程序 任意一方发布新的爆款公式 / 创作者激励政策修订
- **新型红线出现:** 网信办执法中出现本 catalog 未覆盖的新型违规模式(如新型诱导消费、新型深度伪造)
- **受众结构反转:** 男频 / 女频受众比例发生显著反转(如男频受众占比从 60% 降至 40%),需重新评估各类钩子的优先级
- **🔴 条目变化:** 某个原 🔴 条目因平台政策放宽可降级为 🟡,或原 🟡 条目因新规收紧升级为 🔴
- **降级方案失效:** 某个降级方案因平台审核收紧而不再有效(如"商业报复"也被纳入暴力敏感词)
- **新平台入局:** 新兴短剧平台(如 B站短剧、小红书短剧)进入市场,其爆款公式与现有三大平台差异显著

> 本 catalog 的所有钩力保留百分比均为基于公开观察的主观估算(`*estimated*`)。实际钩力需通过平台 A/B 测试验证,超出本 ref 的纯文档范围。
