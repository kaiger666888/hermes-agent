## Taxonomy

下表列出 5 类固定钩子(固定 —— 不允许凭空新增类别)。每个类型给出 一句定义 + [男频](../../_shared/glossary.md#男频--male-oriented-channel)典型应用 + [女频](../../_shared/glossary.md#女频--female-oriented-channel)典型应用。最后一列标注该类型是否与 Phase 1 catalog 重叠。

| 类型 | 定义(一句) | 男频典型应用 | 女频典型应用 | 与 COMPLI catalog 关系 |
|------|-------------|--------------|--------------|------------------------|
| **情感钩** (emotional hook) | 以情感共鸣(亲情 / 友情 / 爱情 / 牺牲)在开场 3 秒抓住观众 | 父子和解 / 兄弟背叛 / 战友情深 | 母女和解 / 替身的爱 / 临终告别 | 重叠 (情感钩) |
| **悬念钩** (suspense hook) | 以未解之谜 / 未发生之事制造持续追看欲 | 主角死亡后重生 / 神秘电话 / 突然消失的家人 | 失忆重逢 / 来历不明的孩子 / 锁着的房间 | HOOK-only (COMPLI 最近 = 角色钩) |
| **冲突钩** (conflict hook) | 以激烈的人际冲突(商战 / 阶级 / 复仇)开场制造紧张 | 商战阴谋 / 阶级碰撞 / 复仇归来 | 婆媳冲突 / 妯娌争斗 / 闺蜜反目 | 重叠 (冲突钩) |
| **反差钩** (contrast hook) | 以角色身份 / 处境的强烈反差制造好奇 | 霸总装穷 / 草根暴富 / 学霸变学渣 | 千金落难 / 丑女逆袭 / 假戏真做 | 重叠 (反差钩) |
| **情绪爆点钩** (emotional-peak hook) | 以单帧 / 单句 / 单音效直接击穿观众情绪阈值的极致瞬间 | 战神归来的第一句台词 / 含冤者终于怒吼 / 兄弟为兄弟挡刀 | 母亲最后一句"妈妈爱你" / 女主决意复仇的转身 / 替身被叫出真名 | HOOK-only (COMPLI 最近 = 题材钩) |

> **关于"重叠 vs HOOK-only":** Phase 1 [`viral-element-catalog.md`](../../compliance_marketing/references/viral-element-catalog.md) 站在"合规视角"组织 5 类(情感 / 冲突 / 反差 / 题材 / 角色);HOOK 站在"设计视角"组织 5 类。设计视角的 悬念 / 情绪爆点 强调"如何制造瞬间注意力",与合规视角的 角色钩 / 题材钩(强调"谁来承载 / 是什么题材")互补。两者不矛盾,只是切片不同。

## 3-Second Frame Structure

每个 3 秒钩子必须按照以下三段式设计。这是 [CONTEXT D-2](../../../../../.planning/phases/02-expert-hook-commercial-engine/02-CONTEXT.md) 决议的固定结构 —— 不允许只设计"开场感觉"而不拆帧。

### 0-1s Attention-Grab

**目的:** 在第一秒制造视觉 / 听觉 / 信息冲击,阻止观众上滑。这一秒的成功率直接决定 [完播率](../../_shared/glossary.md#完播率--completion-rate) 上限。

可用模式(选其一,不可叠加 —— 叠加会信息过载):

- **突发动作 (sudden motion):** 摔门 / 推搡 / 跑动 / 抓手。画面必须有明确运动矢量,不可静态。
- **强音效 (audio sting):** 玻璃碎 / 重低音"咚" / 摔打声 / 突然寂静(对比反差)。注意:抖音小程序剧默认静音播放,音效钩子需配合字幕才能生效。
- **震撼画面 (shocking image):** 血迹 / 婚纱破裂 / 离婚证 / 病危通知 / 倒在地上的孩子。画面本身必须传递情绪信息,不依赖前情。
- **反常规切入 (anti-conventional cold open):** 直接从冲突高潮中段开始 / 跳过铺垫直接给最后一句台词 / 从一个无人能理解的镜头开场。

**反模式 (avoid):** 静态 logo / 慢推大远景 / 长字幕流 / 旁白念背景介绍 —— 这些在 0-1s 几乎必死(对应 5 级评分中的 💀 anti-hook)。

### 1-2s Context-Establish

**目的:** 在第二秒用 "一句话等价信息" 让观众知道 谁 / 在哪 / 利害是什么。这一秒失败 = 观众虽然被 0-1s 抓住,但 2 秒后仍不知道"我在看什么",照样上滑。

可用模式:

- **一句话交代角色处境:** 字幕或台词直接给出 "离婚当天,她才发现自己怀孕了" 这类信息密度极高的句子。
- **场景标识:** 婚礼 / 医院 / 法庭 / 葬礼 / 豪车 —— 一帧即可让观众秒懂情境。
- **关系标识:** 一个称呼("老公" / "妈" / "老板")配合演员反应,确立双方关系。
- **利害锚定:** 必须在这一秒内让观众知道"如果失败会失去什么" —— 孩子的抚养权 / 一家公司 / 一条命 / 一份尊严。

**反模式:** 多角色并行介绍 / 复杂背景设定 / 旁白讲公司历史 / 多条字幕同时滚动。

### 2-3s Hook-Pin

**目的:** 第三秒明确"承诺"观众一个回报 —— 即"看下去你会得到什么"。这是把"被吸引的注意力"转化为"愿意继续看"的关键一秒。

可用模式:

- **明确承诺后续 [爽点](../../_shared/glossary.md#爽点--satisfaction-beat):** "三天后,她让他们全部跪下" / "下一秒,总裁跪在了她面前"。
- **抛出未解之谜:** "但她不知道,真正的危险才刚刚开始" / "而那个电话,来自一个已经死了三年的人"。
- **倒计时感:** "距离她失去一切,还有 24 小时" / "三天后,所有人都将后悔"。
- **反差承诺:** "而当他们发现真相时,已经太晚了" —— 暗示后续会有重大反转。

**反模式:** 模糊承诺("接下来发生了很多事") / 直接剧透结局 / 无承诺的镜头延续。

## Catalog: 15 Examples

以下 15 个示例覆盖 5 类钩子 × 3 示例,且分布于 romance / revenge / family / mystery / comedy 五种题材(每种至少 2 次,单类型内不重复同题材)。每个示例给出固定 5 字段:`示例` / `题材` / `0-1s` / `1-2s` / `2-3s` / `完播率 预期影响`。

所有 [完播率](../../_shared/glossary.md#完播率--completion-rate) 数值均为 `*estimated*`(基于公开 MCN 报告与创作者访谈的聚合观察,不是平台实测数据)。

### 情感钩 (emotional hook)

**示例 1:母女病房重逢**

- **题材:** family
- **0-1s:** 特写女儿推开病房门的颤抖的手 + 监护仪"嘀——"的长音突然响起(突发动作 + 强音效组合;但只用一种主导冲击:监护仪长音)。
- **1-2s:** 字幕打出 "她出差三年,回来时妈妈已经认不出她了" —— 一句话锚定关系与利害。
- **2-3s:** 妈妈转头,茫然问 "你是谁?" —— 明确承诺后续情感爆发(认回或永远失去)。
- **完播率 预期影响:** 提升前 3 秒留存 ~20-30% *estimated*(family 题材在中老年女频受众中粘性最强)。

**示例 2:替身的爱 · 真相之夜**

- **题材:** romance
- **0-1s:** 女主整理男主衣柜时,一张与本人几乎一模一样的旧照片从西装内袋飘落(震撼画面 + 慢镜头)。
- **1-2s:** 镜头切到床头柜合照 —— 那张脸不是她,但和她九成像。一句字幕:"原来我不是他爱的人,我只是一个影子。"
- **2-3s:** 男主脚步声在门外响起,女主把照片塞回原位,转身装作无事 —— 承诺后续揭穿 / 对决爆发。
- **完播率 预期影响:** 提升前 3 秒留存 ~25-35% *estimated*(女频虐恋核心 [爽点](../../_shared/glossary.md#爽点--satisfaction-beat)前置触发器)。

**示例 3:兄弟酒馆挡刀**

- **题材:** revenge
- **0-1s:** 雨夜小巷,主角被三人围住,玻璃酒瓶砸下瞬间,一只手挡在主角头侧 —— 血顺着那只手滴下(突发动作 + 震撼画面组合;主导 = 血滴特写)。
- **1-2s:** 镜头上摇,挡刀的是十年前绝交的兄弟,他低声说 "兄弟的事,什么时候轮到别人插手" —— 一句话锚定关系与利害。
- **2-3s:** 兄弟回头看了主角一眼,眼神复杂 —— 承诺后续兄弟重逢 / 误会解开 / 联手复仇。
- **完播率 预期影响:** 提升前 3 秒留存 ~15-25% *estimated*(男频复仇 + 兄弟情双触发)。

### 悬念钩 (suspense hook)

**示例 4:死亡后第一通电话**

- **题材:** revenge (重生题材)
- **0-1s:** 黑屏。一声突兀的手机铃声响起,主角猛地睁眼 —— 镜头是他自己的葬礼现场,而他正站在人群里看着自己的遗像(反常规切入 + 震撼画面组合)。
- **1-2s:** 字幕:"距离他'死亡',已经过去 72 小时。" —— 一句话锚定时间与利害。
- **2-3s:** 一个陌生人走到他身边,低声说:"他们都以为你死了 —— 这是我们的机会。" —— 抛出未解之谜 + 承诺后续复仇。
- **完播率 预期影响:** 提升前 3 秒留存 ~30-40% *estimated*(悬念 + 重生双重男频钩子,极强)。

**示例 5:锁着的房间**

- **题材:** mystery
- **0-1s:** 新婚女主在新家走廊尽头发现一扇从外面挂了三把锁的门,锈迹斑斑(震撼画面 + 反常规切入)。
- **1-2s:** 字幕:"结婚三年,她从未被告知这扇门后面是什么。" —— 一句话锚定关系与未知。
- **2-3s:** 她伸手碰锁的瞬间,门后传来一声孩子的轻笑 —— 抛出未解之谜,承诺后续揭穿。
- **完播率 预期影响:** 提升前 3 秒留存 ~20-30% *estimated*(女频悬疑 + 母性焦虑双触发)。

**示例 6:失忆重逢 · 路口错身**

- **题材:** romance
- **0-1s:** 繁忙十字路口,男主和女主擦肩而过 —— 两人同时回头,但只看到对方背影(突发动作:回头动作)。
- **1-2s:** 字幕:"五年前那场车祸,带走了她的记忆,也带走了他们之间的一切。" —— 一句话锚定前情。
- **2-3s:** 男主停在原地,低声说了一句她曾经最爱的口头禅 —— 女主脚步顿住,但她不知道为什么 —— 承诺后续记忆唤醒 / 真相。
- **完播率 预期影响:** 提升前 3 秒留存 ~15-25% *estimated*(情感 + 悬念组合,男女通吃)。

### 冲突钩 (conflict hook)

**示例 7:董事会逼宫**

- **题材:** revenge (商战)
- **0-1s:** 董事会会议桌,主角站在尽头,十几个董事同时把辞职信推到他面前(突发动作:同步推信)。
- **1-2s:** 字幕:"他们以为逼他下台,公司就是他们的了。" —— 一句话锚定利害。
- **2-3s:** 主角笑了一下,把手机投影到会议室大屏 —— 屏幕上显示所有人的资金流水 —— 承诺后续打脸 / 翻盘。
- **完播率 预期影响:** 提升前 3 秒留存 ~25-35% *estimated*(男频慕强核心钩子,与 [慕强](../../_shared/glossary.md#慕强--power-fantasy) 爽点直接对接)。

**示例 8:婆媳饭桌掀桌**

- **题材:** family
- **0-1s:** 婆婆把儿媳盛的汤直接倒回锅里,汤汁溅到儿媳白衬衫上(突发动作:倒汤)。
- **1-2s:** 字幕:"结婚五年,她从来没有在婆婆家上过桌吃饭。" —— 一句话锚定关系与利害。
- **2-3s:** 儿媳抬头,平静地说:"妈,从今天开始,这个家我说了算。" —— 承诺后续逆袭 / 规则改写。
- **完播率 预期影响:** 提升前 3 秒留存 ~20-30% *estimated*(女频家庭冲突核心,转发率高)。

**示例 9:职场电梯打脸**

- **题材:** comedy
- **0-1s:** 电梯门开,新入职员工端着咖啡撞到 CEO,咖啡泼了一身(突发动作:碰撞)。
- **1-2s:** 字幕:"第一天上班,她就泼了全公司最不能得罪的人。" —— 一句话锚定情境与利害。
- **2-3s:** CEO 没有发火,反而盯着她看了三秒,说 "你长得像我一个故人" —— 承诺后续身份反转 / 隐藏剧情。
- **完播率 预期影响:** 提升前 3 秒留存 ~15-25% *estimated*(comedy 题材粘性较弱,但隐藏剧情承诺能拉长追看)。

### 反差钩 (contrast hook)

**示例 10:霸总装穷 · 煎饼摊**

- **题材:** romance
- **0-1s:** 烟火气十足的街边煎饼摊,西装革履的男人熟练地摊煎饼、刷酱、卷薄脆(反常规切入:角色与场景强烈错位)。
- **1-2s:** 字幕:"千亿总裁装穷三个月,只为测试她是否爱的是他这个人。" —— 一句话锚定反差与利害。
- **2-3s:** 女主路过,递给他一张皱巴巴的十块钱,说"老板,加个蛋" —— 男主眼神闪烁 —— 承诺后续身份暴露 / 真情考验。
- **完播率 预期影响:** 提升前 3 秒留存 ~25-35% *estimated*(女频反差 + 浪漫双触发,小程序剧核心 [卡点](../../_shared/glossary.md#卡点--paywall-cliffhanger--paywall-moment) 转换器)。

**示例 11:千金落难 · 后厨洗碗**

- **题材:** comedy
- **0-1s:** 米其林餐厅后厨,纤尘不染的千金大小姐戴着橡胶手套,笨拙地刷盘子,一只盘子"啪"地摔碎(突发动作:摔碎)。
- **1-2s:** 字幕:"三天前,她还是身价百亿的女继承人;现在,她时薪 25 块。" —— 一句话锚定反差与利害。
- **2-3s:** 主厨走过来,把一张合同拍在她面前:"签了它,你的债就清了" —— 承诺后续身份反转 / 隐藏目的。
- **完播率 预期影响:** 提升前 3 秒留存 ~15-25% *estimated*(comedy + 反差组合,适合轻松向短剧开场)。

**示例 12:学霸坠落 · 高考放榜**

- **题材:** revenge (人生重启)
- **0-1s:** 高考放榜日,曾经年级第一的主角看到自己名字排在榜单最末位,手中奶茶"啪"地掉在地上(突发动作:掉落)。
- **1-2s:** 字幕:"一年前,他是省状元;今天,他考了倒数第一。" —— 一句话锚定反差与利害。
- **2-3s:** 一个陌生人走过来,递给他一张卡片:"你想重来一次吗?代价是,放弃你现在拥有的一切。" —— 抛出未解之谜 + 承诺后续逆袭。
- **完播率 预期影响:** 提升前 3 秒留存 ~20-30% *estimated*(反差 + 重生悬念组合,青年向男频)。

### 情绪爆点钩 (emotional-peak hook)

**示例 13:战神归来 · 第一句话**

- **题材:** revenge
- **0-1s:** 黑屏。一声低沉的男声:"五年了,我回来了。" 紧接着是机场跑道上私人飞机降落的轰鸣(强音效 + 反常规切入)。
- **1-2s:** 字幕:"五年前他被陷害入狱;五年后,他带着整个帝国的资源回来。" —— 一句话锚定利害。
- **2-3s:** 接机大厅,曾经陷害他的人如今是行业龙头 —— 两人目光相遇,对方脸色骤变 —— 承诺后续碾压式 [爽点](../../_shared/glossary.md#爽点--satisfaction-beat)。
- **完播率 预期影响:** 提升前 3 秒留存 ~30-40% *estimated*(男频战神归来核心钩子,小程序剧付费转换最强触发器之一)。

**示例 14:母亲最后的语音**

- **题材:** family
- **0-1s:** 主角在地铁里戴上耳机,微信语音自动播放 —— 母亲的声音突然响起:"宝贝,妈妈这次可能撑不过去了。"(强音效:语音消息提示音 + 内容冲击)。
- **1-2s:** 字幕:"这条语音,是她在母亲过世后才发现的。" —— 一句话锚定时间错位与利害。
- **2-3s:** 主角猛地站起来,地铁摇晃,她跌坐在地,语音继续:"妈妈有件事瞒了你二十年……" —— 抛出未解之谜,承诺后续真相 + 情感爆发。
- **完播率 预期影响:** 提升前 3 秒留存 ~25-35% *estimated*(family 题材情绪爆点 + 悬念组合,中老年女频转发率极高)。

**示例 15:替身被叫出真名**

- **题材:** mystery
- **0-1s:** 上流社会酒会,女主端着香槟,一个老人突然抓住她的手腕,颤抖地喊出一个陌生的名字(突发动作:抓手 + 强信息冲击)。
- **1-2s:** 字幕:"所有人都以为她只是个普通的小助理 —— 直到今晚。" —— 一句话锚定反差与利害。
- **2-3s:** 女主猛地抽回手,香槟杯"啪"地碎在地上,她的眼神从慌乱瞬间变得冰冷 —— 承诺后续身份揭穿 / 复仇 / 隐藏势力。
- **完播率 预期影响:** 提升前 3 秒留存 ~20-30% *estimated*(mystery + 反差 + 情绪爆点三重叠加,但需注意信息密度,不可再加更多元素)。

## 5-Tier Strength Scoring

下表定义 5 级钩子强度评分(符号固定,不允许替换)。每个候选钩子必须用此表评估后才可投入生产。

| 符号 | 名称 | 预测 [完播率](../../_shared/glossary.md#完播率--completion-rate) 持有区间 | 典型特征 | 修复建议 |
|------|------|--------------------------------------|----------|----------|
| 🎯 | **bullseye** (一击命中) | ≥ 85% 持有 *estimated* | 0-1s 即给观众明确冲击;1-2s 一句话锚定利害;2-3s 承诺具体可感的后续回报。观众生理上无法上滑。 | 无需修复。可作为同题材其他短剧的开场模板。 |
| ✅ | **strong** (强力) | 60-85% 持有 *estimated* | 三段式结构完整,但其中一段稍弱(如 0-1s 冲击到位但 2-3s 承诺略模糊)。能钩住大多数观众,但敏感观众会在 5-8s 流失。 | 定位"最弱的一帧",针对性补强(通常是 2-3s 承诺过于抽象)。 |
| ⚠️ | **weak** (弱钩) | 30-60% 持有 *estimated* | 三段式中只有一段到位。常见:0-1s 有冲击,但 1-2s 信息密度太低 / 2-3s 无承诺。慢热型,需要 pre-roll 预告片引导才能起量。 | 必须重新设计至少 2 段;通常意味着题材本身需要更换钩子类型(如情感钩 → 冲突钩)。 |
| ❌ | **broken** (失效) | < 30% 持有 *estimated* | 三段式结构断裂。典型:开场静态 logo + 长字幕 + 旁白;或开场信息密度过高(同时给 5 个角色介绍);或 0-1s 是与剧情无关的空镜。 | 推倒重做。识别失败的根因(信息过载 / 缺冲击 / 无承诺),按失败类型套用对应的修复路径(见下方自检清单)。 |
| 💀 | **anti-hook** (反向钩子) | 主动驱动上滑 *estimated* | 不仅没钩住,还主动触发观众反感。典型:冷开场 logo 显示 > 2s / 长静音段 / 与平台调性严重冲突的内容(如抖音放 5 分钟慢综艺开场)。 | 立即下线。这种情况通常意味着整个开场设计逻辑错误,不是修复能解决的,需重新立项。 |

### 如何应用本评分表

5 级评分本质上是一个**主观代理指标**(subjective proxy metric)—— [完播率](../../_shared/glossary.md#完播率--completion-rate) 的真实实测需要平台 A/B 测试,超出本 ref 的纯文档范围。但以下结构化流程可让评分尽可能可复现:

1. **逐帧评估:** 把候选钩子的 0-1s / 1-2s / 2-3s 三段分别打分(每段:到位 / 部分到位 / 失效)。
2. **短板原则:** 总分 = 三段中**最低**的一段分数。不允许"0-1s 极强 + 2-3s 失效"被评为 ✅ strong —— 这种钩子实际表现为"瞬间吸引但秒级流失",应评 ⚠️ weak。
3. **题材校准:** 同样的设计在不同题材下表现不同。例如"母女病房重逢"在 family 题材是 🎯,在 revenge 题材可能是 ✅(信息密度足够但情绪指向与题材预期错位)。
4. **平台校准:** 抖音 / 快手 / 微信小程序剧 的观众容忍度不同。同一段开场在抖音可能 ✅,在微信小程序剧可能 ⚠️(小程序剧观众更愿意看长铺垫)。具体平台规则跨链 [Phase 1 platform-specs](../../compliance_marketing/references/) 三份 ref。
5. **风险校准:** 本表 **不评估** 合规风险。即便评分 🎯,若设计涉及 🔴 红线(参见 [`viral-element-catalog.md`](../../compliance_marketing/references/viral-element-catalog.md) 风险徽章),整个钩子必须替换或降级。

### 自检清单(从 ⚠️ 升级到 ✅)

- [ ] 0-1s 是否有单一主导冲击?(不是多重叠加)
- [ ] 1-2s 是否在 1 句话等价信息内同时回答了"谁 / 在哪 / 利害是什么"?
- [ ] 2-3s 是否承诺了**具体可感**的后续回报?(不是"接下来会发生很多事"这种模糊承诺)
- [ ] 三段是否有逻辑因果链?(0-1s 的冲击 → 1-2s 的情境 → 2-3s 的承诺,应是同一个故事的不同切片)
- [ ] 是否避免了所有 [反模式](#0-1s-attention-grab)?

若任一项答"否",该钩子无法达到 ✅ strong,需针对性修复。

## Drift Signals

以下事件触发本 ref 的**非常规(提前)复审** —— 不必等到 90 天周期:

- **平台推荐算法修订:** 抖音 / 快手 / 微信小程序剧 任意一方修订推荐算法,影响 3 秒钩子在完播率权重中的占比(例如:算法从"3 秒留存加权"转向"5 秒留存加权"会重写本 ref 的所有数字)。
- **新爆款 3 秒模式出现:** top 10 创作者社区中出现本 ref 5 类钩子未覆盖的新型 3 秒模式(例如:互动剧形式引入"用户选择即钩子"的新模式)。需评估是否新增第 6 类(谨慎 —— taxonomy 扩容需经过 CONTEXT D-2 类似的决策流程)。
- **生成模型行为漂移:** AI 视频 / 图像 / 音频生成模型的可控性提升或下降,影响 0-1s attention-grab 的可生成性。例如:某模型不再能稳定生成"血滴特写",则"兄弟酒馆挡刀"示例需重新设计。
- **题材结构反转:** 男频 / 女频 / 中老年向 受众占比发生显著反转,导致现有 15 示例的题材分布不再代表主流(例如:中老年向 family 题材占比从 30% 降至 10%,需减少 family 示例、增加其他题材)。
- **跨平台互通:** 若 抖音 / 快手 / 微信小程序剧 的短剧互通(分发机制统一),则平台校准逻辑需重写。

> 本 ref 所有 [完播率](../../_shared/glossary.md#完播率--completion-rate) 数值与"题材粘性"判断均为基于公开观察的主观估算(`*estimated*`)。实际效果需通过平台 A/B 测试验证,超出本 ref 的纯文档范围。
