## Four Core Concepts

### 1. The Gaze (凝视) — from Laura Mulvey

**Foundational essay:** "Visual Pleasure and Narrative Cinema" (1975)

**Core propositions:**
- Cinema contains "scopophilia" (看的快感)
- **Male gaze vs female image:**
  - Male = bearer of the look (subject)
  - Female = form of being looked at (object)
- Three gazes in cinema:
  1. Camera → scene
  2. Audience → screen
  3. Character → character

**Canonical analysis subjects:** Hitchcock (*Rear Window*, *Vertigo*), Sternberg/Dietrich partnership

### 2. Mirror Stage & Double Identification (from Baudry)

**Core propositions:**
- Screen = Lacan's "mirror"
- Audience identifies with the "transcendental subject"
- Double identification:
  - Identify with screen character (imaginary register)
  - Identify with camera's "omniscience" (symbolic register)

### 3. Suture (缝合)

**Core propositions:**
- Films "suture" audiences into narrative via shot/reverse-shot
- Audience's absence is "compensated" by the film
- Suture failure produces "alienation effect" (Brechtian distance)

### 4. Sinthome (圣状) — Žižek on Hitchcock

**Core propositions:**
- Hitchcock's "sinthome" = symptomatic core element in *Vertigo*, *The Birds*
- Sinthome binds the subject's psychic structure
- Remaking Hitchcock requires preserving the sinthome

## Žižek's Lacanian Triad (from 好莱坞中的拉康)

### The Three Registers

| Register | Chinese | Function |
|---|---|---|
| **The Real** | 实在界 | That which cannot be symbolized — the traumatic kernel |
| **The Symbolic** | 象征界 | Language / social order / law |
| **The Imaginary** | 想象界 | Images / mirrors / identifications |

### Žižek's Analytic Path

1. **Find the Real symptom** — e.g., the birds in *The Birds*
2. **Analyze multiple endings** — e.g., *The Matrix* has 3 readings
3. **Notice the "missing gaze"** — what is looked at but not seen
4. **Beware "ideological fantasy"** — *The Matrix*'s "red pill" itself

## Application to AI 短剧

### 短剧 Gaze Structure
Most 短剧 (especially 男频) are **explicitly Mulvey-style**:
- Camera objectifies female characters
- Male protagonist is bearer of the look
- 爽点 = restoration of male gaze power

### Compliance implications
- Platform censorship increasingly flags male-gaze shots
- 女频 短剧 invert gaze (woman as bearer)
- 现代短剧 increasingly use "shared gaze" (both characters look + look back)

### Mulvey test for `compliance_marketing`
1. Identify the bearer of the look in each scene
2. If always one gender → potential platform flag
3. Suggest alternating bearers in re-cut

### Sinthome design for `hook_retention`
- Each 短剧 IP should have a "sinthome" — a recurring symptomatic element
- Example: 战神归来 短剧's sinthome = the moment of recognition
- Identify sinthome → reinforce in opening 3s hook

## Limits of Psychoanalytic Theory for 短剧

- Over-academic for runtime-constrained formats
- Hard to translate "Lacanian lack" into actionable production
- Better as *diagnostic* than *generative* framework
