  - Sync projection
  - Music and SFX
  - Mixing

  - Negative cutting

### Complete 6-stage workflow

#### Stage 1: Development
**Tasks:**
- Find creative (创意制片完全手册 1.1)
- Study script (1.2)
- Plan flow (1.3)
- Obtain adaptation rights (1.4)
- Script creation and revision (1.5)

**Deliverables:**
- Story synopsis
- Script first draft
- Project proposal
- Budget framework

#### Stage 2: Pre-Production
**Tasks:**
- Assemble core creative team
- Casting (audition, contract)
- Location scout (exterior + interior)
- Art design (scene, costume, makeup)
- Equipment list
- Shooting plan (storyboard, shot list)
- Crew recruitment

**Deliverables:**
- Storyboard
- Shooting schedule
- Casting completed
- Makeup/costume locked
- Equipment list

#### Stage 3: Production
**Tasks:**
- Daily shot list
- On-set lighting
- Shot execution
- On-set playback
- Sync sound recording
- Art (scene, props, makeup, costume)
- Stills photography
- Production log (continuity)

**Daily workflow:**
```
5:30 - Core crew arrives, check equipment
6:00 - Lighting setup
6:30 - First scene rehearsal
7:00 - Action!
12:00 - Lunch
13:00 - Continue shooting
18:00 - Wrap
19:00 - Next day prep meeting
```

#### Stage 4: Post-Production
**Tasks (电影制片手册 Ch.4):**
1. **Editing**
   - Material organizing
   - Rough cut
   - Fine cut
   - Picture lock
2. **Sync projection**
3. **Music and SFX**
   - Music composition/selection
   - ADR (dialogue replacement)
   - Foley
   - SFX design
4. **Mixing**
   - Sound synthesis
   - Multi-version (5.1, 7.1, IMAX)
5. **Subtitles and optical effects**
6. **Negative cutting** (film era)
7. **Composite printing** (film era)

#### Stage 5: Color & Finishing
**Tasks:**
- Color grading (DI: Digital Intermediate)
- Output (DCP, IMAX, HDR)
- QC

#### Stage 6: Distribution & Marketing
**Tasks:**
- Festival submission
- Theatrical release
- Streaming release
- International sales
- Marketing materials (poster, trailer, key art)

## Part 4: Low-Budget Filmmaking (英国影视制作基础教程, Project -061)

### Book positioning
UK filmmaking beginner's authoritative guide

### Verified structure
- 造梦 (Dream-making)
- 为什么从短片做起？(Why start with short films?)
- 行动起来吧 (Get moving)
- 公众的需求 (Public needs)
- 短片时代的到来 (Short film era arrives)
- 相机部件 (Camera parts)

### Core proposition: Start with short films

**Book's argument:**
- Short film is "small practice" for feature
- Learn full pipeline
- Low trial cost
- Easier to complete

### 7 steps from zero to one

#### Step 1: Dream (造梦)
- Find the story you "must tell"
- Not "want to tell" but "must tell"
- Write down the image that haunts you

#### Step 2: Write short film script
- **Duration:** 3-15 min optimal
- **Scenes:** 1-3
- **Characters:** 1-3
- **Conflict:** single, clear

**Short film rules:**
- Don't pack too much
- One core creative idea
- One clear ending

#### Step 3: Pre-production (minimal)
- Actors: friends / family
- Locations: real-world places
- Equipment: phone / DSLR / borrowed

#### Step 4: Shooting (2-3 days)
- Day 1: hardest scene
- Day 2: other scenes
- Day 3: pickups

#### Step 5: Editing (self-taught)
- Software: iMovie / Premiere / DaVinci Resolve (free)
- V1: rough cut (build order)
- V2: fine cut (adjust rhythm)
- V3: sound (music + SFX)

#### Step 6: Finish & release
- Optional color grade
- Output (mp4 HD)
- Release (YouTube / Bilibili / Vimeo)
- Submit to festivals

#### Step 7: Review & next
- Watch your own work
- Find strengths and weaknesses
- Write what you learned
- Start next one

### Short film era characteristics
- Digital: extremely low shooting cost
- Free post software
- Many distribution channels
- Mobile audience

### Camera parts 101 (5 core components)
1. **Lens:** image quality
2. **Sensor:** light-sensitive element
3. **Viewfinder:** composition
4. **Storage:** SD card
5. **Battery:** endurance

### Beginner equipment
- **Phone:** iPhone / Huawei flagship (sufficient)
- **Mirrorless:** Sony A7 / Panasonic GH series
- **Cinema camera:** Canon 5D series

## Application to AI 短剧

### `production` expert: workflow adaptation
- AI 短剧 doesn't have "shooting day" → adapt to "generation day"
- 6-stage workflow maps to AI pipeline:
  1. Development = IP selection + script
  2. Pre-production = style genome + storyboard
  3. Production = image + video generation
  4. Post = editing + sound + color
  5. Finishing = platform export
  6. Distribution = 抖音 / 快手 / 小程序剧

### Low-budget workflow for AI 短剧
- "Start with short" = start with 60s episode before 180s
- 7 steps map to AI:
  1. Dream = IP choice
  2. Script = screenplay expert output
  3. Pre-prod = style_genome + scene_builder
  4. Generation = visual_executor (drawer + animator sub-steps)
  5. Post = editor + composer + mixer
  6. Release = platform-specific export
  7. Review = analytics + iteration

### Hollywood model for AI platforms
- 5 platforms (抖音 / 快手 / 小程序剧 / B 站 / 视频号) ≈ 5 studios
- Each has different "house style"
- 短剧 production companies are like subsidiaries
