#### Ch.4: Metal Halide Lamps operation

### Lamp type matrix

#### Tungsten Halogen (钨丝卤素)
- Color temperature: 3200K (warm white)
- Long life (halogen cycle)
- Power range: 150W-10kW+

| Power | Use | Typical |
|---|---|---|
| 150W-650W | Small / fill | Fresnel small spot |
| 1kW-2kW | Medium / key | Arri 650/1000/2000 |
| 5kW-10kW | Large / base | Tener, Tweni |
| 10kW+ | Very large / night exterior | Skypan, fan light |

**Use precautions:**
- Never touch bulb with bare hands (oil corrodes)
- Hot during operation — do not touch
- Verify voltage before switching

#### Metal Halide / HMI
- Color temperature: 5600K (daylight)
- High luminous efficiency (4x brighter than tungsten)
- Requires ballast

| Power | Use | Typical |
|---|---|---|
| 200W-575W | Small daylight | Joker |
| 1.2kW-2.5kW | Medium daylight | Mole 1.2k |
| 4kW-6kW | Large daylight | Mole 4k/6k |
| 12kW-18kW | Very large | Musco, Cranestone |

**Use precautions:**
- 5-10 second startup delay
- Ballast must be reliably grounded
- Voltage instability causes flicker

#### LED (modern supplement)
- Energy-efficient, low heat
- Variable color temperature
- Very long life

**Types:**
- Panel (Astra, Skypanel)
- Tube (LightTractrix)
- High-power (Vortex 8, Cineo 350)

### 5 Light Position Types

#### 1. Key Light
- Main light source
- Usually 45° to one side of subject
- Determines overall light feel

#### 2. Fill Light
- Balances key's shadow
- Usually on opposite side
- Determines contrast

#### 3. Back Light / Rim Light
- Outlines silhouette
- Usually behind and above
- Separates subject from background

#### 4. Background Light
- Lights the background
- Provides layers
- Creates atmosphere

#### 5. Practical / Effects Light
- Visible on-set sources (bulbs, candles, screens)
- Adds realism

### Classic Lighting Patterns

#### Three-point lighting (basic)
- Key + Fill + Back
- Suitable for close-up

#### Rembrandt lighting (dramatic)
- Single source + half-face shadow
- Inverted triangle cheek highlight
- Suitable for dramatic scenes

#### Butterfly / Paramount lighting (female)
- Top light + dual side fill
- Butterfly shadow under nose
- Suitable for female close-up

#### Split lighting (high contrast)
- Single side, half-lit half-shadowed
- Strong drama
- Suitable for villain / inner struggle

### Safety operation

#### Electrical safety
- Calculate total load (A = W / V)
- Three-phase balance
- RCD required
- Reliable grounding

#### Aerial work
- Sandbag the stand
- Secure high stands (rope/chain)
- Lower in wind

#### High-temperature protection
- Never touch bulb during operation
- Cool 5 min before switching
- Wear thermal gloves

#### Chapter 3: Light

- 理性写实主义 (Rational realism)

#### Chapter 4: Space / Environment / Shot Scale
- Color in different spaces / shot scales

### Color Narrative's 3 Layers

#### Layer 1: Scene Color
Each scene has a primary color:
- Red scene = passion / danger
- Blue scene = melancholy / alienation
- Yellow scene = nostalgia / warmth
- Green scene = sickness / unease

#### Layer 2: Character Color
Each main character has a color signature:
- Protagonist red (passion) vs villain blue (cold-blooded)
- Character arc expressed through color change

#### Layer 3: Emotional Color
**Book's key line (verified):**
> "情绪：高潮是红的，叹息是蓝的" (Emotion: climax is red, sigh is blue)

Meaning: Color directly corresponds to emotional beat.

### "Color within Color" & "Flat Montage"

#### Color within color
**Definition:** A scene nests a "color substructure" within itself.
- Example: Red room with blue clothing — red vs blue conflict
- Example: B&W film with single colored object — focal reinforcement

#### Flat montage
**Definition:** Meaning produced by color "cutting", not picture cutting.
- Example: Shot A red → Shot B red = "color equivalence" of two events
- Example: Cold to warm cut = emotional transition

### Rational Realism

**Book quote (verified):**
> "理性写实主义：冷静残酷的摄影态度" (Rational realism: cool, cruel photographic attitude)

Meaning:
- No sentimentality
- Objective gaze
- Let cruelty speak for itself
- Representative: Jia Zhangke early works, Fifth Generation mid-period

### Classic Color Narrative Cases

| Film | Color Strategy |
|---|---|
| *Hero* | Segmented colors (red/blue/green/white) |
| *In the Mood for Love* | Warm nostalgia + shadow |
| *Crouching Tiger, Hidden Dragon* | Natural color + wuxia poetry |
| *Raise the Red Lantern* | Red-black-white contrast |
| *Happy Together* | B&W and color transition |
| *Call Me By Your Name* | Italian summer warmth |
| *Her* | Warm tone + high contrast |
| *Moonlight* | Three segments, three color tones |

### Color arc design
```
Scene 1: cold blue (setup) →
Scene 2: neutral (development) →
Scene 3: warm orange (emotional rise) →
Scene 4: extreme cold (conflict) →
Scene 5: warm red (climax) →
Scene 6: white (release)
```

## Part 4: 物像 / 视像 / 影像 / 画像 (from 影视光线艺术, Project -063)

### Four-image theory (verified from book structure)

| Image Type | Definition | Function |
|---|---|---|
| **物像** (Object image) | The actual physical object | What exists |
| **视像** (Visual image) | What the eye sees | What's perceived |
| **影像** (Cinematic image) | What the camera captures | What's recorded |
| **画像** (Picture image) | The final on-screen picture | What audience sees |

### Relationship chain
```
Object (物像) → Perception (视像) → Capture (影像) → Picture (画像)
```

### Cinematographer's task
- Recognize each stage differs
- Manipulate the translation between stages
- Use light as primary expressive tool

## Application to AI 短剧

### Equipment selection for AI 短剧
- AI 短剧 usually doesn't shoot live → equipment mainly for reference plates
- For live-action 短剧: LED panels + natural light (低成本 priority)
- Specify equipment list in `production`'s GPU budget

### Color narrative for `colorist`
- Beyond LUT — narrative-driven color
- Each 短剧 episode has a "color arc" matching beat structure
- "Color within color" = emphasize single object in desaturated scene

### Production design for `scene_builder`
- Build visual bible per 短剧 IP
- Per-scene color script
- 6-stage workflow adapted to 短剧 runtime
