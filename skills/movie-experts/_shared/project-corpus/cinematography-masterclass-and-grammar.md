  - 3.2 Documentary film

### Three Core Methods

#### Method 1: Parallel Cutting (Ch.2)

**Two basic types (2.1):**
- **Type A:** Two main lines happening simultaneously, eventually meeting
  - Example: *High Noon* — sheriff and outlaws' parallel lines
- **Type B:** One main line + one psychological line
  - Example: *Godfather* ending baptism scene

**Key principle: Action and Reaction (2.2)**
- Every action must have a reaction
- "Action" without "reaction" = failed shot
- Reaction can come from: another character, environment, audience

**Classic examples (2.4):**
- **Chase on New York Street:** Chased / chaser alternate cutting
- **Dancer:** Dancer / audience / judges triangular alternation

#### Method 2: Peak Moment Theory (2.3)

**Definition:**
- A scene's most intense moment of emotion / action
- Don't shoot entire process, only shoot "peak"
- Use "peak moment + reaction" instead of "complete action"

**Examples:**
- Boxing match: don't shoot entire fight, only shoot punch moment + fall moment + audience reaction
- Wedding: don't shoot entire ceremony, only shoot ring exchange + kiss + parents' reaction

#### Method 3: Three Basic Types Scene Handling (Ch.3)

| Type | Shooting Principle | Cutting Principle |
|---|---|---|
| **News film** | Objective, candid, no intervention | Chronological, key frames |
| **Documentary film** | Long-term observation, theme focus | Theme-driven, point-of-view |
| **Story film** | Designed, blocked, controllable | Dramatic effect, emotional rhythm |

### Parallel Cutting's 5 Relationship Types (from Ch.2.4)

1. **Simultaneous:** Two events happen at same time
2. **Causal:** A causes B
3. **Contrast:** A and B contrast
4. **Metaphorical:** A metaphor for B
5. **Psychological:** External action vs internal psychology

### Classic Case: *High Noon*'s parallel cutting
```
Line A: Sheriff Kane prepares to fight (in town)
Line B: Outlaws arrive at station (outside town)
 ↓
Line A: Kane seeks help, refused
Line B: Outlaws gather
 ↓
Line A: Kane faces alone
Line B: Outlaws enter town
 ↓
Convergence: Showdown
```

**Rhythm control:**
- Closer to convergence point → faster cutting
- Time pressure increases
- Psychological pressure intensifies

## Part 2: 100 Cinematic Storytelling Techniques (Project -035)

### Book positioning
"Cinematic Storytelling: The 100 Most Powerful Film Techniques" — Jennifer van Sijll

### Verified classification system

| Category | Subcategories | Typical Techniques |
|---|---|---|
| **Space** | Location / Shot scale / Composition | Use location to express character identity |
| **Light** | Bright / Dark / Contrast | Use shadow to hint at character inner |
| **Sound** | Diegetic / Non-diegetic / Non-narrative | Use silence to create tension |
| **Time** | Order / Flashback / Juxtaposition | Use flashback to reveal trauma |
| **Movement** | Camera / Character | Use tracking shot for immersion |
| **Dialogue** | Rhythm / Omission / Monologue | Use subtext to express conflict |
| **Object** | Props / Symbols | Use object to link theme |

### Core proposition
- **Literary storytelling:** uses words
- **Cinematic storytelling:** uses audiovisual elements
- **Screenwriter's task:** convert literary thinking to audiovisual thinking

### Selected techniques

#### Space techniques
- Location as character extension (*Godfather* dark office vs bright home)
- Public vs private space contrast (*American Beauty*)
- Stairs / levels as power metaphor (*Metropolis*)

#### Light techniques
- Shadow = inner darkness (*Apocalypse Now* Kurtz's face)
- High-key vs low-key (comedy vs film noir)
- Single-source drama (candle, flashlight)

#### Sound techniques
- Silence as drama tool (*Apocalypse Now* opening)
- Diegetic vs non-diegetic
- Sound bridge (A's sound extends to B)

#### Time techniques
- Flashback as trauma reveal (*Citizen Kane* "Rosebud")
- Flashforward for suspense
- Simultaneity for irony (*Godfather* baptism vs revenge)

#### Movement techniques
- Tracking = immersion (*Russian Ark* single shot)
- Push-in = psychological entry
- Pull-out = alienation

#### Dialogue techniques
- Subtext ("Coffee is good" = "I love you")
- Overlapping dialogue (Altman)
- Monologue vs dialogue

#### Object techniques
- Prop linking theme (*Citizen Kane* Rosebud / sled)
- Object transformation (*2001* bone → spaceship)

### Application to 短剧
- Every scene should use at least one cinematic technique
- Critical emotion should NOT use dialogue
- Use space / light / sound / object to convey information
- When adapting from novel, convert to cinematic technique

### Hot Opening vs Cold Opening

#### Hot opening
**Definition:** Opens with action / suspense / conflict
**Structure:**
```
Raid (hook) → Conflict establishment → Information reveal → Main plot
```

#### Cold opening
**Definition:** Opens with emotion / dialogue / atmosphere
**Structure:**
```
Atmosphere establishment → Character introduction → Theme hint → Main plot
```

## Part 4: 21 Cinematographers' Master Class

### From 光影创作课 (Project 076)

**Verified structure:**
- **Part 1: Cinematography Technology Basics**
- **Part 2: Light Sources** (core)
  - Ch.4 Single light source (035)
  - Ch.5 Two hard light sources (045)
  - Ch.6 Hard light and soft light (057)
  - Ch.7 Moving light source (071)

### 7 Light Source Handling Methods

#### 1. Single light source (Ch.4)
- One light does everything
- Suitable: night scenes, interior, dramatic
- Cases: Kubrick *Barry Lyndon* candlelight; Almendros *Days of Heaven* dusk

#### 2. Two hard light sources (Ch.5)
- Key + back (no fill)
- Suitable: male characters, dramatic
- Advantage: high contrast, sculptural

#### 3. Hard light and soft light (Ch.6)
- **Hard light:** direct, clear shadows (Rembrandt, film noir)
- **Soft light:** scattered, no clear shadows (rom-com, advertising)
- **Mix:** soft key, hard back

#### 4. Moving light source (Ch.7)
- Light follows subject
- Simulate fire, car headlights
- Cases: flashlight, candle

#### 5. Natural light (sunlight)
- Use windows, doors
- Reflectors, flags
- Cases: *Godfather* interior natural light

#### 6. Practical light
- Visible on-set sources
- Bulbs, lanterns, screens
- Cases: *Blade Runner* practical lights

#### 7. Mixed light sources
- Different color temperatures
- Create atmosphere
- Cases: *Magnolia* multi-source

### Roger Deakins methodology (from 导演的摄影课)

**Core proposition:**
> "Serve the story, not the visual"

**Characteristics:**
- Prefers natural light, practical light
- Skilled with LED, windows
- Anti-over-artificial lighting

**Representative works:**
- *No Country for Old Men* — desert sun
- *Blade Runner 2049* — neon mix
- *1917* — long-take war

### Néstor Almendros methodology (from 光影大师)

**Core proposition (verified chapter title):**
> "Use fewer lights?" "How many?"

**Characteristics:**
- Minimalism
- Prefers magic hour
- Skilled with reflectors

**Representative works:**
- *Days of Heaven* — magic hour
- *Kramer vs. Kramer* — natural interior
- *The Last Metro* — vintage film look
