## Part 1: Disney 4-Stage History (Project 【迪士尼】)

### Book positioning
"The authoritative study of Disney art and development"
**Author:** Christopher Finch, 5th illustrated edition

### Verified structure
- **Part 1: Early Career (早期事业)** — 1920s
- **Part 2: Animation Features (动画长片)** — 1937+ (Snow White onward)
- **Part 4: Live-Action Films (实景拍摄影片)** — 1950s
- **Part 5: Magic Kingdom (魔幻王国)** — 1955+ (Disneyland onward)

### Stage 1: Early Career (1920s)
- **1923:** Walt arrives in Hollywood
- **1928:** *Steamboat Willie* — first Mickey Mouse with sound
- **1929-1939:** *Silly Symphonies* series — technical experimentation
  - Color (*Flowers and Trees* 1932)
  - Multiplane camera (*The Old Mill* 1937)
- **1934:** Donald Duck debut
- **1937:** *Snow White* — first feature-length animation

**Stage 1 lessons:**
- Technical experimentation precedes artistic maturity
- Short-form innovation → feature-length mastery
- Recurring IP (Mickey) funds experimentation

### Stage 2: Animation Features (1937+)
- **Golden Age (1937-1942):** *Snow White*, *Pinocchio*, *Fantasia*, *Bambi*, *Dumbo*
- **Wartime / Package Era (1942-1949):** *Saludos Amigos*, *Three Caballeros* — lower budget
- **Silver Age (1950-1967):** *Cinderella*, *Peter Pan*, *Sleeping Beauty* — Walt's lifetime
- **Bronze Age (1970-1988):** *Aristocats*, *Robin Hood* — post-Walt decline
- **Disney Renaissance (1989-1999):** *Little Mermaid*, *Beauty and the Beast*, *Aladdin*, *Lion King*
- **Post-Renaissance (2000-2009):** Mixed results
- **Revival Era (2010+):** *Tangled*, *Frozen*, *Moana*, *Encanto*

**Stage 2 lessons:**
- Animation features go through cycles
- Renaissance driven by Broadway-style musical integration
- Revival driven by cultural specificity (Scandinavian / Polynesian / Colombian)

### Stage 3: Live-Action Films (1950s)
- **1950:** *Treasure Island* — first fully live-action
- **1954:** *20,000 Leagues Under the Sea*
- **1961:** *The Parent Trap*, *101 Dalmatians* (live-action/animation hybrid)
- **1964:** *Mary Poppins* — animation/live-action integration peak
- **Nature documentaries:** *True-Life Adventures* series (1948-1960)

**Stage 3 lessons:**
- Live-action diversifies studio revenue
- Hybrid (animation + live-action) is Disney's signature
- Nature documentaries = early Disney+ DNA

### Stage 4: Magic Kingdom (1955+)
- **1955:** Disneyland opens (Anaheim)
- **1971:** Disney World (Orlando)
- **1983:** Tokyo Disneyland
- **1992:** Disneyland Paris
- **2005:** Hong Kong Disneyland
- **2016:** Shanghai Disney Resort

**Stage 4 lessons:**
- Theme parks = IP monetization pinnacle
- Geographic expansion = cultural adaptation
- Park experience ≠ film experience — different design principles

## Part 2: 12-Stage Animation Production Pipeline

### Standard Disney animation pipeline

#### Stage 1: Story Idea (故事构思)
- 1-sentence pitch
- Source selection: folk tale, fairy tale, original
- Studio green-light process

#### Stage 2: Script (故事剧本)
- 90-120 pages
- Includes song placement markers
- Multiple revisions (often 10+ drafts)

#### Stage 3: Storyboard (故事板)
- Whole film drawn as storyboard
- One image per beat
- Full studio review ("pitch session")

#### Stage 4: Voice Casting (配音)
- Record voice BEFORE animation
- Voice actor's performance drives animation timing
- Celebrity voice = marketing value

#### Stage 5: Leica Reel (故事板影片)
- Storyboard + voice → test rhythm
- Adjust cuts, add/delete scenes
- Often multiple iterations

#### Stage 6: Character Design (角色设计)
- Model sheet (multiple angles)
- Construction guide
- Scale reference

#### Stage 7: Art Direction (美术设计)
- Color script (per-scene color)
- Background style
- Overall visual identity

#### Stage 8: Key Animation (原画)
- Senior animators draw key frames
- Define acting, timing, expression
- Often 1 senior per character

#### Stage 9: In-betweens (中间画)
- Junior animators fill frames between keys
- Mechanical, time-consuming
- Often outsourced

#### Stage 10: Backgrounds (背景)
- Separate from character animation
- Painted (2D) or modeled (3D)
- Multi-layer for depth

#### Stage 11: Coloring / Compositing (上色 / 合成)
- **2D:** ink-and-paint (digital today)
- **3D:** rendering (compute-intensive)

#### Stage 12: Post-Production (后期)
- Music recording (orchestral)
- SFX design
- Final mix
- Color grading

## Part 3: Cross-Cultural Adaptation — *Mulan* Case (Project -062)

### Book chapter (verified)
- 花木兰 Mulan
- "看迪斯尼如何演绎中国古代传说" (How Disney interprets Chinese ancient legend)

### Mulan adaptation comparison

| Source Element | Disney Adaptation |
|---|---|
| 北朝民歌 (Northern Dynasty folk poem) | 90-min feature animation |
| Filial piety (孝道) as core | Individual value (个人价值) as core |
| Sparse narrative | Musical comedy |
| No sidekick character | Mushu dragon + Cri-Kee cricket + horse Khan |
| Chinese battlefield | American-style action set-pieces |
| No romantic subplot | Shang romantic interest |
| Mulan accepts her role | Mulan "finds herself" |
| Tragic undertones | Triumphant ending |

### Cross-cultural adaptation 5 rules

#### Rule 1: Find universal motif
- *Mulan*: family honor + self-discovery
- Universal = global audience can resonate

#### Rule 2: Add American values
- *Mulan*: freedom, individual pursuit of destiny
- Original: filial duty, family above self

#### Rule 3: Create sidekick characters
- Mushu dragon = Eddie Murphy comedy
- Cri-Kee cricket = "lucky" gag
- Sidekicks regulate pace + add humor

#### Rule 4: Design musical numbers
- "Reflection" — Mulan's identity crisis
- "I'll Make a Man Out of You" — training montage
- "A Girl Worth Fighting For" — army marching
- Songs = emotional punctuation

#### Rule 5: Preserve cultural elements as decoration
- Architecture: Forbidden City inspired
- Costume: Han dynasty inspired
- Color: red/gold/green Chinese palette
- Calligraphy, dragon imagery

## Part 4: Three Animation Types

### 2D Animation
- Classic Disney style
- High handcraft warmth
- Cases: *Snow White*, *Lion King*, *Spirited Away* (Ghibli)

### 3D / CG Animation
- Pixar style
- Realistic lighting, rich detail
- Cases: *Toy Story*, *Up*, *Inside Out*

### Stop Motion
- Physical objects shot frame-by-frame
- Handcraft texture
- Cases: *Chicken Run*, *Coraline*, *Kubo*

## Part 5: Musical Storytelling Principles

### When to use song
- **Identity moments** — "Reflection" (*Mulan*), "Let It Go" (*Frozen*)
- **Training / transformation montages** — "I'll Make a Man Out of You"
- **Group celebration** — "Under the Sea" (*Little Mermaid*)
- **Villain declaration** — "Poor Unfortunate Souls"
- **Romantic duet** — "A Whole New World" (*Aladdin*)

### Song function in plot
- **I want song** — opening (sets protagonist's desire)
- **Showstopper** — midpoint (largest production number)
- **Climax song** — emotional peak
- **Finale** — resolution / theme statement

### Song integration rules
- Song MUST advance plot, not pause it
- Song lyrics = subtext externalized
- Tempo of song = pace of scene
- Visuals in song = metaphorical expansion

## Part 6: Sidekick Comedy Structure

### Protagonist + Sidekick formula
- **Protagonist:** audience identification
- **Sidekick:** audience laughter + relief
- Sidekick often breaks 4th wall or comments on protagonist

### Classic Disney sidekicks
- *Mulan*: Mushu + Cri-Kee
- *Aladdin*: Genie + Abu + Magic Carpet
- *Little Mermaid*: Sebastian + Flounder
- *Lion King*: Timon + Pumbaa
- *Frozen*: Olaf + Sven

### Sidekick design rules
- Visually contrasting (smaller / weirder than protagonist)
- Voiced by comedian (Eddie Murphy, Robin Williams)
- Often non-human (animal, magical being)
- Provides comic relief in heavy scenes

## Application to AI 短剧 / Animation 短剧

### Animation 短剧 production via Hermes
- 12-stage pipeline compressed for 短剧:
  - Stage 1-2: AI generates story + script (1 day)
  - Stage 3: storyboard via `drawer` (1 day)
  - Stage 4-5: voice via `voicer` + leica reel (1 day)
  - Stage 6-7: character + color script via `drawer` + `colorist` (1 day)
  - Stage 8-11: `animator` execution (veo3.1 / kling) (1-2 days)
  - Stage 12: `composer` + `mixer` (1 day)
- Total: 6-8 days for 60-180s animation 短剧

### Disney style DNA in `style_genome`
For animation 短剧 IP, encode Disney DNA:
- Density: 8-9 (Disney is dense)
- Formality: 6-7 (structured but expressive)
- Color: 7-9 (color script heavy)
- Motion: 5-6 (varied but not chaotic)
- Sound: 8-10 (musical heavy)

### Cross-cultural adaptation for 短剧 IP
- Many 短剧 IP adapt Chinese folk tales (西游记 / 红楼梦 / 武侠)
- Apply 5 rules:
  - Universal motif: love, revenge, destiny
  - Modern values: gender equality, individual choice
  - Sidekick: 必备滑稽角色
  - Musical: opening hook as "song"
  - Cultural decoration: 视觉文化元素
