## Summary

本 ref 是 **v1 题材锚定的权威源**。除非 operator 显式覆盖,**所有 v1 生产默认假设本题材**(都市奇幻·轻喜剧)。题材锚定决定了核心 DNA、per-platform 内容形态、3-month 启动路线图、变现逻辑 与 题材禁忌。

任何 v1 项目启动前必须先查阅本 ref,确认:(1) 项目题材是否落在锚定范围内;(2) 目标平台的内容形态规格;(3) 当前所处启动期月份(M1/M2/M3)对应的工作项。若 operator 显式选择其他题材(例如 都市言情 / 玄幻修真),必须显式标注"题材覆盖 v1 默认",并放弃本 ref 中所有 per-platform 形态与 monetization 数据。

## 核心 DNA (Core DNA)

> Reproduced verbatim from Notion §剧集策略 §核心 DNA。3 条 DNA 是 v1 题材的不变量,任何 v1 制作必须三者全在。

- **超能力设定 (superpower setting)** —— 角色拥有可被视觉化的超自然能力(读心 / 隐身 / 时间停滞 等),作为差异化识别的视觉锚点(参见 `creative-redlines.md` R5 差异化识别)。超能力必须在每集至少出现 1 次显性展现。
- **轻喜剧 (light comedy)** —— 整体基调以喜剧 / 反差 / 解构为主,不追求沉重;情绪脱敏(参见 `creative-redlines.md` R1)天然被喜剧节奏打断。悲剧 / 重口 / 写实暴力 均不属于本 DNA(见 §题材禁忌)。
- **主线悬念 (ongoing mystery)** —— 跨集主线问题(主角的超能力从何而来 / 谁是幕后 / 最终目标),与每集单集冲突并行;每集结尾未完成(参见 `creative-redlines.md` R4)必须挂钩到主线。

## 启动方案 3-Month Roadmap

> Reproduced verbatim from Notion §启动方案 §3-month roadmap。3 个子阶段(M1 / M2 / M3)对应 v1 启动期 3 个月工作流。bullets 为 Notion 原文 verbatim。

### M1 验证 (Validation)

- **目标:** 在 B站 / 抖音 双平台验证"超能力设定 + 轻喜剧"题材的市场反应;N≥10 样本(参见 `creative-redlines.md` R7 统计显著)。
- **输出:** 5 集 B站季番正剧(5-10 分钟)+ 抖音超能力高光切片(每集 3-5 条切片)。
- **KPI:** 完播率(B站)/ 5 秒留存(抖音);N≥10 才做题材 / 节奏 决策。
- **不作:** 不做周边 / 不做条漫改编 / 不做视频号长版(M2/M3 再做)。

### M2 平台适配 (Platform Adaptation)

- **目标:** 基于 M1 验证结果,扩展至 5-7 平台;针对每平台做 per-platform cut 差异化。
- **输出:** 视频号长版 + 片尾能力解读;小红书角色穿搭图集;红果·快手 1-2min 快节奏版;腾讯漫画·快看条漫改编启动。
- **KPI:** per-platform 完播率 / 关注转化率;A/B 实验进入正式循环(每实验遵守 `creative-redlines.md` R6 控制变量)。
- **不作:** 不做周边(M3 再做);不做 IP 延伸。

### M3 IP 延伸 (IP Extension)

- **目标:** 基于 M2 稳定的 5-7 平台流量,启动周边 / 数字藏品 / 联名美妆 变现。
- **输出:** 淘宝·小红书店铺 周边(3D 手办 / 数字藏品 / 联名美妆);跨平台 IP 整合营销(联动短片 / 跨平台彩蛋)。
- **KPI:** 周边 GMV / IP 授权收入 / 联名品牌数;单 IP 总收入目标 50-200 万(参见 §变现逻辑)。
- **不作:** 不开新题材 IP(本 IP 稳定运营 ≥ 6 个月后再考虑第二 IP)。

## Per-Platform 变现逻辑 (Monetization Logic)

> Reproduced from Notion §启动方案 §变现逻辑。平台 / 机制 / 关键系数 三列。数值为 Notion 原文,直接 monetization 决策依据。

| 平台 (Platform) | 变现机制 (Monetization Mechanism) | 关键系数/单价 (Key Coefficients) |
|-----------------|-----------------------------------|-----------------------------------|
| 红果 | 仿真人系数(短剧分成) | 仿真人系数 ≈ 60(单位:元 / 万播放,具体以平台规则为准) |
| 抖音 / 快手 | 广告分账(短剧分成 + 流量包) | 广告分账 CPM ≈ 30-50 元(单位:元 / 千展示,具体以平台规则为准) |
| B站 | 创作激励 + 充电 + 大会员专属 | 单剧收入估算 50-200 万元(中位数,头部可达更高;具体取决于完播 / 追番 / 充电转化) |
| 小红书 | 品牌合作 + 店铺带货 | 单条品牌合作报价 5000-50000 元(看账号等级);店铺 GMV 看周边 SKU |
| 视频号 | 广告分账 + 微信小程序剧付费 | 单剧 50-200 万元(与 B站量级接近;小程序剧付费解锁分账 60-70%) |
| 淘宝·小红书店铺(周边) | 周边 GMV + 数字藏品 + 联名美妆 | 单 IP 周边 GMV 上限取决于 IP 热度,典型 10-100 万元;数字藏品毛利率 80%+ |

**读法提示:**
- 所有数值为 Notion 原文;**仅作 monetization 决策参考,不作合同 / 财报依据**。平台规则变化时需重新对照 Notion 源(参见 `platform-specs.md` Refresh Cadence 同样的季度复检原则)。
- 单 IP 总收入 = sum(各平台收入);目标 50-200 万区间(B站单剧 + 抖音分账 + 视频号 + 周边 累计)。

## 题材禁忌 (Genre Taboos)

> **Derived list — NOT verbatim from Notion.** 本节是从都市奇幻·轻喜剧 DNA 反推的题材禁忌,每条解释为何违反 DNA。

- **重口狗血 (heavy melodrama)** —— 违反"轻喜剧"DNA。重口冲突(自残 / 极端虐待 / 极端复仇)与轻喜剧基调冲突,会导致喜剧节奏断裂 + 观众预期错位 → 脱粉。
- **悲剧收尾 (tragic ending)** —— 违反"轻喜剧"DNA 与 `creative-redlines.md` R4(结尾未完成)。悲剧收尾本身就是 closure,而本 DNA 要求每集结尾埋钩子,悲剧基调使后续"反转 / 续集"失去说服力。
- **设定堆砌 (lore dumps)** —— 违反"轻喜剧 + 主线悬念"DNA。大段世界观 / 超能力系统 exposition 会触发 `creative-redlines.md` R3(零背景铺垫)+ 稀释悬念(主线应被埋藏,不可一次性交代)。
- **写实暴力 (realistic violence)** —— 违反"超能力设定"DNA 与 `compliance_gate` §2(暴力血腥)。写实暴力与超能力的"非现实"基调冲突,且极易触发法律红线;超能力展现应优先 stylized / 抽象化表达。
- **genre drift 至玄幻修真 (genre drift to xianxia/cultivation)** —— 违反"都市"DNA。玄幻修真(上古秘境 / 修真体系)与"都市"设定冲突,会导致 per-platform 内容形态失效(都市奇幻 fit 短剧 5-10min,玄幻修真 fit 长剧集 30+min)。若 operator 显式选择本题材,必须显式覆盖 v1 默认。
- **genre drift 至都市言情 (genre drift to urban romance)** —— 违反"超能力设定"DNA。纯都市言情(无超能力 / 无悬念主线)会丢失差异化识别(`creative-redlines.md` R5)+ 失去跨平台 IP 延伸能力(周边 / 数字藏品 fit 超能力视觉锚点,fit 不了纯言情)。若 operator 显式选择本题材,必须显式覆盖 v1 默认。

**读法提示:**
- 这些禁忌是 DNA 反推,**非法律禁令**(法律的在 `compliance_gate`)。违反任一即题材偏离,不会触发合规 fail,但会削弱 v1 题材策略的可执行性。
- 若 operator 显式选择其他题材(覆盖 v1 默认),禁忌表自动失效 —— 该题材有其自身的禁忌结构。

## Why This Genre for V1 (Rationale)

> Reproduced / derived from Notion §第一性原理分析。3 个理由解释为何 v1 选择都市奇幻·轻喜剧。

- **(1) FLUX 角色一致性测试用例 (FLUX character consistency test case)** —— 超能力的视觉签名(发光眼 / 浮空物件 / 时间停滞特效)是强 character anchor,能在 FLUX / 类图像生成模型上做严苛的角色一致性测试。如果模型能在每集保留超能力视觉签名,就能保留更复杂的角色细节。都市奇幻·轻喜剧是 FLUX 测试的"理想压力测试"。
- **(2) 3D 资产 IP 延伸路径 (3D asset IP extension path)** —— 都市奇幻的超能力视觉锚点天然适合做 3D 手办 / 数字藏品 / 联名美妆(M3 启动);纯都市言情或都市职场剧无强视觉锚点,周边变现路径窄。本题材让 v1 在 M3 阶段有明确的 IP 延伸可能性,降低单平台依赖风险。
- **(3) 多平台复用 (multi-platform reuse)** —— 一个 IP → 8 平台内容形态(per-platform 表),降低 per-platform 内容成本。都市奇幻·轻喜剧同时 fit B站长版(主线推进)+ 抖音短切片(超能力爆点)+ 小红书美学(超能力视觉)+ 周边变现(超能力道具),是一种"高 ROI 题材";窄题材(纯悬疑 / 纯职场)只能 fit 1-2 平台。

**第一性原理 (first principles) 框架:**
v1 启动期的核心问题不是"做什么题材最好",而是"做什么题材能让 v1 投入产出比最高"。都市奇幻·轻喜剧在 (a) 技术可行性(FLUX 一致性)+ (b) 变现延伸(周边)+ (c) 平台复用(8 平台)三个维度同时占优,是 v1 阶段唯一能在 3 个月内验证完整 IP 闭环的题材。
