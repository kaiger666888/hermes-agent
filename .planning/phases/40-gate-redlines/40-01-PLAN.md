Detector result shape (introduced by this plan):
```python
# plugins/review_gates/gates/types.py
from __future__ import annotations
from typing import Literal, Protocol

Decision = Literal["approve", "reject", "contest"]
SuggestedAction = str | None
DetectorResult = tuple[Decision, SuggestedAction]

class DetectorFn(Protocol):
    """Detector protocol — pure function over clip beat-sequence payload.

    Detectors MUST NOT import httpx / jwt / yaml / plugins.* (D-34-01 extended
    to redline detectors — same rationale as gate.py: unit-testable in
    isolation, no network/config coupling).
    """
    GATE_ID: str   # e.g. "redline_emotion_desensitize"
    REDLINE_REF: str  # e.g. "creative-redlines.md §R1"
    def detect(self, payload: dict) -> DetectorResult: ...
```

Clip payload shape (input — emitted by upstream pipeline Phase 3 script_audit):
```python
payload = {
    "episode_id": "EP01",
    "beats": [
        {
            "index": 0,
            "label": "active_conflict",      # one of: active_conflict / exposition / narration /
                                              # setup / resolution / closure / epilogue / other
            "emotion": "anger",              # emotion taxonomy tag (R1 input)
            "duration_sec": 2.5,
            "text": "She slammed the door...",  # optional beat text
        },
        ...
    ],
    "platform": "douyin_vertical",           # affects 3s/10s thresholds
    "runtime_sec": 55.0,
}
```
