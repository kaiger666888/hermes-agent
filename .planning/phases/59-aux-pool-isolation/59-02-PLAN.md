Build POOL-02: per-key TPM tracking in the auxiliary pool so `acquire_slot` picks the freshest key (most remaining TPM) and warns when all keys are running low.

       class TpmWindow:

           TPM_WINDOW_SECONDS: ClassVar[float] = 60.0

           @classmethod
           def fresh(cls) -> "TpmWindow":
