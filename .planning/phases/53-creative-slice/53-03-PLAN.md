## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| operator cli-config.yaml → mcp_serve.py::get_agent_opinion | Config controls provider routing; misconfiguration (e.g. provider: openrouter) violates MEMORY.md GLM-only policy. |
| driver script → MCP tool closures | In-process Python call (no stdio transport). Assumes Wave 0 verified closures are async-callable. |
| GLM API → auxiliary_client.call_llm | Network call; timeout / failure possible. Response content untrusted. |
| StoryKernel fixture → driver | Synthetic input; if replaced with operator-curated content, content influences downstream persona prompt. |
