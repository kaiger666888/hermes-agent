# Plan 09-01 — corpus-trace.yaml canonical structure

> **Phase:** 9 of v2.0 PRFP · **Plan:** 09-01 of 2
> **Deliverable:** `.planning/research/v2-pipeline-design/corpus-trace.yaml`
> **Covers CORPUS reqs:** CORPUS-01 (bidirectional matrix), CORPUS-02 (applicable_form), CORPUS-03 (challenge-source), CORPUS-04 (principle/workflow), CORPUS-05 (Chinese terms), CORPUS-06 (AIGC-native flag), CORPUS-07 (Last-verified)

## Goal

A machine-readable YAML file that is the **single source of truth** for corpus ↔ node traceability. `03-CORPUS-TRACEABILITY.md` will render from it.

## YAML Schema (top-level)

```yaml
schema_version: design-2026-06-16-prfp
generated: 2026-06-16
corpus_base:
  total_books: 102
  total_book_files_cited: <count>  # actual files referenced
  primary_corpus_path: skills/movie-experts/_shared/project-corpus/
  book_index_path: <repo>/indexes/books-index.md  # 102-book canonical list
  hermes_integrated_refs_count: 12  # files in project-corpus/
  readiness:
    ready_to_cite_directly: 9
    needs_supplementation: 3  # screenwriting-短剧 / psychoanalytic-短剧 / AIGC-marginal-value

nodes:   # 16 entries (15 linear + 1 consultative)
  - id: <node_id>
    corpus_anchors:        # CORPUS-01 forward direction; CORPUS-02/04/05
      - source_type: hermes_ref | book | external_v1_ref | web
        source_id: <stable id, e.g. "PC-cinematography-masterclass-and-grammar.md">
        source_title: <human title>
        books_synthesized: [<book ids from 102-book index, e.g. -036 拉片子>]
        applicable_form: universal | 长片 | 微电影 | 短剧   # CORPUS-02
        form_translation_note: <if source 长片 but applied to 短剧, why>
        separation:                       # CORPUS-04
          principle: <validated-invariant, e.g. "cut on emotion, not action">
          workflow_obsolete: <anachronistic part, e.g. "review dailies on Steenbeck">
        chinese_terms:                    # CORPUS-05
          - term: 意境
            gloss: mood/atmosphere (Buddhist-aesthetic lineage)
            untranslatable_flag: true
        principle_vs_workflow_separated: true   # CORPUS-04 boolean for lint
        last_verified: 2026-06-16         # CORPUS-07
        citation_strength: strong | medium | weak | zero
    challenge_source:                      # CORPUS-03 — at least 1 per node
      source_id: <id of disagreeing source>
      source_title: <title>
      disagreement_summary: <one-sentence what it challenges>
      design_response: <one-sentence why we keep our design anyway>
      last_verified: 2026-06-16
    aigc_native_flag:                      # CORPUS-06
      value: true | false
      rationale: <if true: why this node has no traditional corpus anchor>
    zero_strength_explanation: <only present if aigc_native_flag.value == true>

books_reverse_index:   # CORPUS-01 reverse direction
  - book_id: <e.g. -036>
    book_title_zh: <汉字 title>
    book_title_en: <English gloss>
    informs_nodes: [<node_ids>]
    applicable_form: <universal | 长片 | 微电影 | 短剧>
    last_verified: 2026-06-16
```

## Required field checks (per node entry)

- ≥1 `corpus_anchors` entry per node (CORPUS-01 forward)
- Every anchor has `applicable_form` (CORPUS-02)
- Every anchor has `separation.principle` + `separation.workflow_obsolete` (CORPUS-04)
- Every anchor has `last_verified` (CORPUS-07)
- Exactly 1 `challenge_source` per node (CORPUS-03) — may be an external (non-corpus) source if no in-corpus challenge exists
- `aigc_native_flag.value == true` nodes MUST have `zero_strength_explanation` (CORPUS-06)
