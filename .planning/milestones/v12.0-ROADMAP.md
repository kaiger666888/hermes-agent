## Key Architectural Decisions (locked from v11.0 smoke-test-report)

1. **Endpoint routing threshold:** 4096 input tokens (configurable)
2. **RPM default:** 30/task (well under GLM single-key ceiling)
3. **TPM budget default:** 100K per round table
4. **Cost formula:** `(tokensConsumed/1M) * 0.5 / 7.2` USD (GLM-5.2 pricing)
5. **Aux pool isolation:** `GLM_AUX_API_KEY_1..4` env vars (fallback to `GLM_API_KEY`)
6. **Per-key TPM cap:** 200K/key (GLM ceiling, hardcoded for v12.0)
7. **TPM warning threshold:** 10% remaining
8. **Freshest-key selection:** `select_freshest_tpm` wired into `_select_pool_entry` for aux pool only (default pool keeps configured strategy)
