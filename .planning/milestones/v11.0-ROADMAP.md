## Key Architectural Decisions (locked from v10.0 design suite)

1. **7 MCP tool names** per REQUIREMENTS.md PoC scope (NOT `02-ROUND-TABLE-PROTOCOL.md §5` broader list)
2. **State enum** `{open, completed, aborted, stalled}` per `round-table-state-schema.yaml` (NOT informal `in_progress/closed` shorthand)
3. **Memory tools** ship as Phase 52 stubs → Phase 53 fills in mem0 routing with `unavailable` graceful fallback
4. **9-agent subset** mapped to v3.0+ current names: `screenplay, cinematographer, hook_retention, theory_critic, editor, character_designer, continuity_auditor, audio_pipeline, style_genome`
5. **Conflict arbitration** full 5-mechanism per `02-ROUND-TABLE-PROTOCOL.md §3` (comparator LLM + scope precedence + confidence voting + tie-deferral + JSONL conflict log)
6. **HOOK-09 invariant** preserved in `screenplay.agent.yaml` `lineage.transform_notes` (literal substring)
7. **Strict serial execution** via per-`roundId` `asyncio.Lock` (NOT threading.Semaphore — that's `agent/glm_concurrency_guard.py`'s job)
8. **Dry-run-first** as Python default value + AST-walk non-bypassable invariant (NOT CLI convention)
