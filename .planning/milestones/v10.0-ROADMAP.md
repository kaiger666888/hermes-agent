## Overview

v10.0 is a **design-only milestone** (zero source code changes, analog to v2.0 PRFP). It produces 7 design docs + 2 validation artifacts that together specify the Hermes-Agent orchestrator + Hermes-native expert agents + Claude Code execution场 three-layer architecture, derived from first principles and informed by Kimi Notion 架构2.0 + the 4 in-repo research threads (STACK / FEATURES / ARCHITECTURE / PITFALLS). The design suite will guide the v11.0 PoC implementation milestone.

**Paradigm shift:** First milestone producing pure design artifacts without code changes since v2.0 PRFP. Output is a citation-chain of markdown documents + JSON Schemas + a Python lint script that together form the v11.0 PoC implementation blueprint.

**Scope:**
- 7 design docs (`00-FIRST-PRINCIPLES` through `06-CROSS-REPO-IMPACT`)
- 3 JSON Schemas (agents-schema, memory-record-schema, round-table-state-schema)
- 1 Python lint script (`scripts/v10-consistency-check.py`)
